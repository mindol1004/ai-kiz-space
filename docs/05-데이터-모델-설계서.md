# 키즈공간 데이터 모델 설계서

## 1. ERD 개요

본 문서는 키즈공간 서비스의 데이터베이스 모델을 정의한다. Prisma ORM을 사용하며 PostgreSQL을 기반으로 한다.

---

## 2. 엔티티 정의

### 2.1 사용자 (User)

| 컬럼 | 타입 | 필수 | 설명 |
|------|------|------|------|
| id | String (cuid) | PK | 사용자 고유 ID |
| email | String | O (unique) | 이메일 |
| password | String | X | 해싱된 비밀번호 (소셜 로그인 시 null) |
| name | String | O | 실명 |
| nickname | String | O (unique) | 닉네임 |
| phone | String | X | 휴대폰 번호 |
| profileImage | String | X | 프로필 이미지 URL |
| role | Enum (USER, ADMIN, SUPER_ADMIN) | O | 사용자 역할 |
| grade | Enum (BRONZE, SILVER, GOLD, VIP) | O | 회원 등급 |
| points | Int | O | 보유 포인트 (기본: 0) |
| provider | Enum (EMAIL, KAKAO, NAVER, GOOGLE) | O | 가입 수단 |
| providerId | String | X | 소셜 로그인 고유 ID |
| agreeMarketing | Boolean | O | 마케팅 수신 동의 |
| status | Enum (ACTIVE, SUSPENDED, WITHDRAWN) | O | 계정 상태 |
| lastLoginAt | DateTime | X | 마지막 로그인 일시 |
| createdAt | DateTime | O | 가입일 |
| updatedAt | DateTime | O | 수정일 |

**관계:**
- 1:N → Child (자녀 정보)
- 1:N → Address (배송지)
- 1:N → Order (주문)
- 1:N → CartItem (장바구니)
- 1:N → Wishlist (위시리스트)
- 1:N → Review (리뷰)
- 1:N → Post (게시글)
- 1:N → Comment (댓글)
- 1:N → Coupon (쿠폰)
- 1:N → Notification (알림)
- 1:N → PointHistory (포인트 이력)

**주문 상태 (OrderStatus):**

```
PENDING        → 주문접수 (결제 대기)
PAID           → 결제완료
PREPARING      → 상품준비중
SHIPPED        → 배송중
DELIVERED      → 배송완료
CONFIRMED      → 구매확정
CANCELLED      → 주문취소
RETURN_REQUEST → 반품신청
RETURNED       → 반품완료
EXCHANGE_REQUEST → 교환신청
EXCHANGED      → 교환완료
```

**관계:**
- N:1 → User
- 1:N → OrderItem
- N:1 → Coupon

## 3. 인덱스 설계

| 테이블 | 인덱스 | 타입 | 용도 |

| User | email | UNIQUE | 로그인, 중복 확인 |
| User | nickname | UNIQUE | 중복 확인 |
| User | provider, providerId | COMPOUND | 소셜 로그인 조회 |

## 4. 데이터 관계도 (요약)

```
User ─────┬──── Child (1:N)
          ├──── Address (1:N)
          ├──── CartItem (1:N) ──── Product (N:1)
          ├──── Wishlist (1:N) ──── Product (N:1)
          ├──── Order (1:N) ──── OrderItem (1:N) ──── Product (N:1)
          ├──── Review (1:N) ──── Product (N:1)
          ├──── Post (1:N) ──── Comment (1:N)
          ├──── Coupon (1:N)
          ├──── PointHistory (1:N)
          ├──── Notification (1:N)
          └──── Bookmark (1:N) ──── Post (N:1)

Category ──── Category (Self 1:N, parentId)
         └──── Product (1:N)

Brand ──── Product (1:N)

Product ──── ProductImage (1:N)
        ├──── ProductOption (1:N)
        └──── Exhibition (M:N)

Exhibition ──── Banner (관련)
```
