# 키즈공간 서비스 개요서

> **작성 기준**: planning-expert 산출물 형식
> **관련 스킬**: planning-expert, architecture-expert, security-expert

---

## 1. 프로젝트 개요

- **서비스명**: 키즈공간 (Kids Space)
- **목적**: 부모와 예비 부모를 위한 키즈 전문 쇼핑몰 겸 커뮤니티 플랫폼. 양질의 아동용품 구매와 육아 정보 공유를 하나의 서비스에서 제공한다.
- **타겟 사용자**: 25~40세 부모 및 예비 부모 (모바일 퍼스트)
- **핵심 가치**: 원스톱 경험 (정보 탐색 → 구매 → 후기 공유의 선순환), 신뢰 기반 커뮤니티, 자녀 연령 맞춤 추천

## 2. 비즈니스 분석

### 2.1 린 캔버스 (Lean Canvas)

| 영역 | 내용 |
|------|------|
| **문제** | 1. 키즈 제품 정보가 분산되어 있어 비교/검증이 어려움 2. 기존 쇼핑몰에 커뮤니티 부재 3. 연령별 맞춤 추천 부족 |
| **고객 세그먼트** | 25~40세 부모/예비 부모, 워킹맘, 전업주부, 예비 아빠 |
| **고유 가치 제안** | "정보 → 구매 → 후기" 선순환 키즈 전문 플랫폼 |
| **솔루션** | 1. 쇼핑+커뮤니티 통합 2. 연령 기반 큐레이션 3. 실사용자 리뷰 검증 |
| **채널** | 모바일 웹/앱, 검색엔진(SEO), 육아 커뮤니티 마케팅, 인플루언서 |
| **수익원** | 상품 판매 마진, 입점 수수료, 광고, 프리미엄 멤버십 |
| **비용 구조** | 서버/인프라, 상품 매입, 물류, 마케팅, 인건비 |
| **핵심 지표** | MAU, 주문 전환율, 객단가, 게시글 작성율, 재방문율, NPS |
| **경쟁 우위** | 쇼핑+커뮤니티 통합, 키즈 전문 큐레이션, 소통 기반 구매 경험 |

### 2.2 SWOT 분석

| | 긍정적 | 부정적 |
|---|--------|--------|
| **내부** | **강점 (S)** | **약점 (W)** |

| **외부** | **기회 (O)** | **위협 (T)** |

| | - 인플루언서 마케팅 효과 | - 저출산 장기 추세 |

## 3. 타겟 사용자 페르소나

|------|------|
| **자녀** | 3세 아들 |
| **특징** | 시간이 부족해 효율적인 쇼핑을 원함, 다른 부모의 후기를 참고하여 구매 결정 |

| **Pain Point** | 여러 사이트를 돌아다니며 리뷰 확인하는 번거로움 |
| **기대 행동** | 모바일에서 통근 중 상품 탐색 → 커뮤니티 후기 확인 → 즉시 구매 |

| 항목 | 내용 |
|------|------|

| **특징** | 육아 용품에 대한 사전 지식이 없어 정보 수집 중 |
| **니즈** | 출산 준비 체크리스트, 추천 상품 가이드, 초보 부모 커뮤니티 |
| **Pain Point** | 어떤 제품을 사야 할지 판단 기준이 없음 |
| **기대 행동** | Q&A 게시판에서 질문 → 추천 상품 확인 → 위시리스트 저장 → 구매 |

## 4. 핵심 기능

| 기능 | 설명 | 우선순위 | 비고 |
|------|------|----------|------|
| 회원가입/로그인 | 이메일, 소셜(카카오/네이버/구글) 로그인 | P0 (Must) | NextAuth.js |
| 사용자 프로필 | 자녀 정보, 관심사 설정, 프로필 관리 | P0 (Must) | 연령 기반 추천 연동 |
| 상품 카탈로그 | 카테고리별 상품 목록, 정렬, 필터링 | P0 (Must) | SSR + 무한스크롤 |
| 상품 상세 | 이미지 갤러리, 옵션 선택, 관련 리뷰 | P0 (Must) | ISR revalidate |
| 상품 검색 | 키워드 검색, 자동완성, 최근/인기 검색어 | P0 (Must) | 디바운스 300ms |
| 장바구니 | 상품 추가/수정/삭제, 비회원 로컬 지원 | P0 (Must) | Zustand 로컬 → 서버 동기화 |
| 주문/결제 | 주문서 작성, PG사 결제 연동 | P0 (Must) | 토스페이먼츠/포트원 |
| 주문 관리 | 주문 내역, 배송 추적, 취소/반품/교환 | P0 (Must) | 상태 전이 관리 |
| 커뮤니티 게시판 | 카테고리별 게시판 (육아팁, 제품후기, Q&A, 자유) | P0 (Must) | Rich Text Editor |
| 댓글 시스템 | 게시글 댓글, 대댓글 (1depth) | P0 (Must) | Q&A 채택 기능 포함 |
| 관리자 대시보드 | 상품/주문/회원/게시글 관리 | P0 (Must) | RBAC 기반 권한 |
| 상품 리뷰 | 텍스트/이미지 리뷰, 평점 | P1 (Should) | 구매 확정 후 작성 |
| 위시리스트 | 관심 상품 저장 및 관리 | P1 (Should) | 토글 방식 |
| 좋아요/북마크 | 게시글/댓글 좋아요, 게시글 북마크 | P1 (Should) | 낙관적 업데이트 |
| 알림 | 주문 상태, 댓글, 이벤트 알림 | P1 (Should) | 인앱 + 이메일 |
| 공지사항/FAQ | 공지사항 및 자주 묻는 질문 | P1 (Should) | SSG |
| 기획전/이벤트 | 시즌별 기획전, 할인 이벤트 | P2 (Could) | ISR |
| 쿠폰 시스템 | 쿠폰 발급, 적용, 관리 | P2 (Could) | 정률/정액 |
| 포인트 시스템 | 구매/활동 포인트 적립, 사용 | P2 (Could) | 적립/사용/만료 이력 |
| 연령별 추천 | 자녀 연령 기반 상품/콘텐츠 추천 | P2 (Could) | 개인화 알고리즘 |

## 5. 사용자 스토리 및 수용 기준

### 5.1 쇼핑 관련

- [ ] **US-001**: As a 부모, I want 자녀 연령에 맞는 상품을 필터링하고 싶다, so that 적합한 제품을 빠르게 찾을 수 있다.
  - [ ] AC-001: 연령대 필터(0~6개월, 6~12개월, 1~2세, 3~5세, 6~8세, 9세 이상) 선택 시 해당 상품만 표시된다
  - [ ] AC-002: 자녀 정보 등록 시 메인 페이지에 해당 연령대 추천 상품이 자동 노출된다
  - [ ] AC-003: 필터 해제 시 전체 상품이 다시 표시된다

- [ ] **US-002**: As a 구매자, I want 다른 부모들의 실제 사용 후기를 보고 싶다, so that 구매 결정에 확신을 가질 수 있다.
  - [ ] AC-004: 상품 상세 페이지에서 리뷰 탭을 통해 평점 분포와 리뷰 목록을 확인할 수 있다
  - [ ] AC-005: 포토 리뷰만 필터링하여 볼 수 있다
  - [ ] AC-006: 리뷰를 최신순/평점 높은순/평점 낮은순으로 정렬할 수 있다

- [ ] **US-003**: As a 바쁜 부모, I want 모바일에서 간편하게 주문하고 싶다, so that 이동 중에도 쇼핑할 수 있다.
  - [ ] AC-007: 모바일 화면에서 모든 쇼핑 플로우가 터치 인터페이스에 최적화되어 있다
  - [ ] AC-008: 모바일 하단 고정 바에서 장바구니 담기/바로 구매가 가능하다
  - [ ] AC-009: 간편결제(카카오페이, 네이버페이, 토스페이) 1~2탭으로 결제 가능하다

- [ ] **US-004**: As a 구매자, I want 장바구니에 담은 상품을 한 번에 결제하고 싶다, so that 여러 상품을 효율적으로 구매할 수 있다.
  - [ ] AC-010: 장바구니에서 개별/전체 선택 후 일괄 주문이 가능하다
  - [ ] AC-011: 품절 상품은 선택 불가 처리되고 "품절" 배지가 표시된다
  - [ ] AC-012: 30,000원 이상 주문 시 무료 배송이 자동 적용된다

- [ ] **US-005**: As a 구매자, I want 주문 상태와 배송 현황을 확인하고 싶다, so that 언제 도착하는지 알 수 있다.
  - [ ] AC-013: 마이페이지에서 주문 상태(주문접수→결제완료→상품준비→배송중→배송완료)를 확인할 수 있다
  - [ ] AC-014: 배송중 상태에서 운송장 번호 클릭 시 택배사 배송 추적 페이지로 이동한다
  - [ ] AC-015: 주문 상태 변경 시 알림이 발송된다

### 5.2 커뮤니티 관련

- [ ] **US-006**: As a 부모, I want 육아 고민을 다른 부모들과 나누고 싶다, so that 경험 기반 도움을 받을 수 있다.
  - [ ] AC-016: 카테고리별(육아팁, 제품후기, Q&A, 자유) 게시판에서 글을 조회할 수 있다
  - [ ] AC-017: 게시글에 댓글과 대댓글(1depth)을 작성할 수 있다
  - [ ] AC-018: Q&A 게시판에서 질문자가 답변을 채택할 수 있다

- [ ] **US-007**: As a 경험 많은 부모, I want 육아 팁과 제품 후기를 작성하고 싶다, so that 다른 부모들에게 도움이 된다.
  - [ ] AC-019: Rich Text Editor로 이미지가 포함된 게시글을 작성할 수 있다
  - [ ] AC-020: 제품후기 게시판에서는 연관 상품을 링크할 수 있다
  - [ ] AC-021: 태그를 최대 5개까지 입력할 수 있다
  - [ ] AC-022: 글 작성 시 10포인트가 적립된다 (일 최대 5건)

- [ ] **US-008**: As a 사용자, I want 관심 있는 게시글을 북마크하고 싶다, so that 나중에 다시 찾아볼 수 있다.
  - [ ] AC-023: 게시글 상세에서 북마크 버튼을 토글할 수 있다
  - [ ] AC-024: 마이페이지에서 북마크한 게시글 목록을 확인할 수 있다

### 5.3 공통

- [ ] **US-009**: As a 신규 사용자, I want 소셜 계정으로 간편하게 가입하고 싶다, so that 빠르게 서비스를 이용할 수 있다.
  - [ ] AC-025: 카카오/네이버/구글 소셜 로그인으로 가입할 수 있다
  - [ ] AC-026: 최초 소셜 로그인 시 닉네임, 휴대폰, 약관 동의 추가 입력이 필요하다
  - [ ] AC-027: 동일 이메일의 기존 계정이 있으면 계정 연동 안내가 표시된다

- [ ] **US-010**: As a 사용자, I want 알림을 받고 싶다, so that 중요한 소식을 놓치지 않는다.
  - [ ] AC-028: 주문 상태 변경, 댓글, 대댓글, 좋아요 시 인앱 알림이 생성된다
  - [ ] AC-029: 헤더에서 읽지 않은 알림 수가 뱃지로 표시된다
  - [ ] AC-030: 알림 설정에서 유형별 수신 여부를 변경할 수 있다

- [ ] **US-011**: As a 관리자, I want 상품과 주문을 효율적으로 관리하고 싶다, so that 원활한 서비스 운영이 가능하다.
  - [ ] AC-031: 관리자 대시보드에서 오늘 매출, 신규 주문, 신규 회원, 미처리 건을 한눈에 확인할 수 있다
  - [ ] AC-032: 상품 등록/수정/삭제/상태변경이 가능하다
  - [ ] AC-033: 주문 상태 변경 및 운송장 입력이 가능하다
  - [ ] AC-034: 모든 관리자 행위는 감사 로그(audit log)에 기록된다
