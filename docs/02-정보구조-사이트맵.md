## 2. URL 구조 설계

### 2.1 퍼블릭 페이지

| URL 패턴 | 페이지 | 렌더링 전략 | SEO |
|----------|--------|------------|-----|
| `/` | 홈 | ISR (60s) | O |
| `/shop` | 상품 목록 | SSR | O |
| `/shop/category/[slug]` | 카테고리별 상품 | SSR | O |
| `/shop/product/[id]` | 상품 상세 | ISR (300s) | O |
| `/shop/search` | 검색 결과 | SSR | X |
| `/shop/best` | 베스트 상품 | ISR (600s) | O |
| `/shop/exhibition` | 기획전 목록 | ISR (600s) | O |
| `/shop/exhibition/[id]` | 기획전 상세 | ISR (300s) | O |
| `/community` | 커뮤니티 메인 | SSR | O |
| `/community/[boardSlug]` | 게시판 목록 | SSR | O |
| `/community/[boardSlug]/[postId]` | 게시글 상세 | ISR (60s) | O |
| `/info/notices` | 공지사항 | SSG | O |
| `/info/faq` | FAQ | SSG | O |
| `/info/terms` | 이용약관 | SSG | O |
| `/info/privacy` | 개인정보처리방침 | SSG | O |
| `/info/about` | 회사소개 | SSG | O |

### 2.2 인증 필요 페이지

| URL 패턴 | 페이지 | 렌더링 전략 |
|----------|--------|------------|
| `/auth/login` | 로그인 | CSR |
| `/auth/register` | 회원가입 | CSR |
| `/mypage` | 마이페이지 | CSR |
| `/mypage/*` | 마이페이지 하위 | CSR |
| `/cart` | 장바구니 | CSR |
| `/checkout` | 주문/결제 | CSR |
| `/community/write` | 글쓰기 | CSR |
| `/admin/*` | 관리자 | CSR |

## 4. 화면 흐름도 (Screen Flow)

### 4.1 상품 구매 플로우

```
[홈/카테고리/검색]
        │
        ▼
  [상품 목록]  ←──── [필터/정렬]
        │
        ▼
  [상품 상세]
    │       │
    ▼       ▼
[장바구니]  [바로구매]
    │           │
    ▼           │
[장바구니 페이지]│
    │           │
    └─────┬─────┘
          ▼
    [주문서 작성]
      │       │
      ▼       ▼
  [결제 진행]  [주소 선택/추가]
      │
      ▼
  [결제 완료] ──▶ [주문 상세]
```

### 4.2 회원가입/로그인 플로우

```
[서비스 접근]
      │
      ▼
  [로그인 필요?] ── No ──▶ [서비스 이용]
      │ Yes
      ▼
  [로그인 페이지]
    │         │
    ▼         ▼
[이메일 로그인] [소셜 로그인]
    │         │   │   │
    │         ▼   ▼   ▼
    │      [카카오][네이버][구글]
    │              │
    ▼              ▼
[회원 존재?] ── No ──▶ [회원가입]
    │ Yes                    │
    ▼                        ▼
[로그인 완료] ◀──── [프로필 설정]
      │                     │
      ▼                     ▼
[이전 페이지] ◀──── [자녀 정보 입력 (선택)]
```

### 4.3 커뮤니티 글쓰기 플로우

```
[커뮤니티]
    │
    ▼
[게시판 선택]
    │
    ▼
[글쓰기 클릭]
    │
    ▼
[로그인 확인] ── 미로그인 ──▶ [로그인 페이지]
    │ 로그인됨
    ▼
[글 작성 에디터]
  │ 카테고리 선택
  │ 제목 입력
  │ 본문 작성 (이미지 첨부)
  │ 태그 입력
    │
    ▼
[미리보기] ──▶ [수정]
    │
    ▼
[게시] ──▶ [게시글 상세]
```

## 5. 콘텐츠 계층 구조

### 5.1 상품 정보 계층

```
상품
├── 기본 정보
│   ├── 상품명
│   ├── 가격 (정가, 할인가)
│   ├── 브랜드
│   ├── 카테고리 (대/중/소)
│   └── 태그
├── 상세 정보
│   ├── 상품 설명
│   ├── 이미지 갤러리
│   ├── 옵션 (색상, 사이즈 등)
│   ├── 상세 스펙
│   └── 적정 연령
├── 구매 정보
│   ├── 재고 상태
│   ├── 배송 정보
│   ├── 반품/교환 정책
│   └── A/S 안내
└── 사용자 생성 콘텐츠
    ├── 리뷰 (평점, 텍스트, 이미지)
    └── Q&A
```

### 5.2 커뮤니티 콘텐츠 계층

```
게시글
├── 메타 정보
│   ├── 게시판 카테고리
│   ├── 작성자
│   ├── 작성일
│   ├── 조회수
│   └── 태그
├── 본문
│   ├── 제목
│   ├── 내용 (Rich Text)
│   └── 첨부 이미지
└── 상호작용
    ├── 좋아요 수
    ├── 댓글
    │   ├── 댓글 내용
    │   ├── 대댓글
    │   └── 좋아요
    └── 북마크
```
