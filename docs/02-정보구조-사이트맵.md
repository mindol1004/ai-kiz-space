## 2. URL 구조 및 렌더링 전략

> **근거**: frontend-expert 렌더링 전략 (SSR/SSG/ISR/CSR 판단 기준), architecture-expert Server/Client Component 구분

### 2.1 퍼블릭 페이지 (SEO 필요)

| URL 패턴 | 페이지 | 렌더링 | 컴포넌트 타입 | SEO | 캐싱 전략 |
|----------|--------|--------|---------------|-----|-----------|
| `/` | 홈 | ISR | Server (배너/상품 fetch) + Client (슬라이더) | O | revalidate: 60s |
| `/shop` | 상품 목록 | SSR | Server (초기 데이터) + Client (필터/정렬/무한스크롤) | O | no-store (동적 필터) |
| `/shop/category/[slug]` | 카테고리별 | SSR | Server + Client | O | no-store |
| `/shop/product/[id]` | 상품 상세 | ISR | Server (상품 정보) + Client (옵션 선택, 리뷰 탭) | O | revalidate: 300s |
| `/shop/search` | 검색 결과 | SSR | Server + Client (자동완성) | X | no-store |
| `/shop/best` | 베스트 | ISR | Server | O | revalidate: 600s |
| `/shop/exhibition` | 기획전 목록 | ISR | Server | O | revalidate: 600s |
| `/shop/exhibition/[id]` | 기획전 상세 | ISR | Server + Client | O | revalidate: 300s |
| `/community` | 커뮤니티 메인 | SSR | Server + Client (탭) | O | no-store |
| `/community/[boardSlug]` | 게시판 목록 | SSR | Server + Client (페이지네이션) | O | no-store |
| `/community/[boardSlug]/[postId]` | 게시글 상세 | ISR | Server (본문) + Client (댓글, 좋아요) | O | revalidate: 60s |
| `/info/notices` | 공지사항 | SSG | Server | O | 빌드 시 생성 |
| `/info/faq` | FAQ | SSG | Server | O | 빌드 시 생성 |
| `/info/terms` | 이용약관 | SSG | Server | O | 빌드 시 생성 |
| `/info/privacy` | 개인정보처리방침 | SSG | Server | O | 빌드 시 생성 |
| `/info/about` | 회사소개 | SSG | Server | O | 빌드 시 생성 |

### 2.2 인증 필요 페이지

| URL 패턴 | 페이지 | 렌더링 | 컴포넌트 타입 | 인증 레벨 |
|----------|--------|--------|---------------|-----------|
| `/auth/login` | 로그인 | CSR | Client | 비인증만 |
| `/auth/register` | 회원가입 | CSR | Client | 비인증만 |
| `/auth/forgot-password` | 비밀번호 찾기 | CSR | Client | 비인증만 |
| `/cart` | 장바구니 | CSR | Client | 선택 (비회원 로컬 지원) |
| `/checkout` | 주문/결제 | CSR | Client | 필수 (USER) |
| `/checkout/complete` | 결제 완료 | CSR | Client | 필수 (USER) |
| `/mypage` | 마이페이지 | CSR | Client | 필수 (USER) |
| `/mypage/*` | 마이페이지 하위 | CSR | Client | 필수 (USER) |
| `/community/write` | 글쓰기 | CSR | Client | 필수 (USER) |
| `/admin` | 관리자 대시보드 | CSR | Client | 필수 (ADMIN) |
| `/admin/*` | 관리자 하위 | CSR | Client | 필수 (ADMIN) |

### 2.3 Middleware 인증/인가 설계

> **근거**: backend-expert 인증/인가 패턴, security-expert RBAC

```
Middleware 규칙:
├── /auth/* → 로그인 상태면 / 로 리다이렉트
├── /mypage/* → 비로그인이면 /auth/login?callbackUrl= 로 리다이렉트
├── /checkout/* → 비로그인이면 /auth/login?callbackUrl= 로 리다이렉트
├── /community/write → 비로그인이면 /auth/login?callbackUrl= 로 리다이렉트
├── /admin/* → role !== ADMIN/SUPER_ADMIN 이면 403 또는 / 로 리다이렉트
└── /api/admin/* → role !== ADMIN/SUPER_ADMIN 이면 403 응답
```

## 3. 네비게이션 구조

> **근거**: designer-expert UX 설계, frontend-expert 반응형 패턴

### 3.1 GNB (Global Navigation Bar) — Desktop

```
┌─────────────────────────────────────────────────────────────────┐
│ [로고: 키즈공간]   [쇼핑 ▾]   [커뮤니티]   [기획전]              │
│                              [검색 🔍]  [장바구니 🛒(n)]  [👤]  │
└─────────────────────────────────────────────────────────────────┘
```

- **로고**: 클릭 → `/` 이동
- **쇼핑 ▾**: 호버 시 메가 메뉴 (카테고리 3depth + 베스트/신상품 바로가기)
- **커뮤니티**: 클릭 → `/community`
- **기획전**: 클릭 → `/shop/exhibition`
- **검색**: 클릭 시 검색바 확장, 자동완성/최근검색/인기검색 드롭다운
- **장바구니**: 뱃지(담긴 수량), 0이면 뱃지 미표시
- **프로필 아이콘**: 비로그인 → [로그인][회원가입] / 로그인 → 드롭다운(마이페이지, 주문내역, 로그아웃)
- **관리자 계정**: [관리자] 버튼 추가 노출

### 3.2 쇼핑 메가 메뉴

```
┌──────────────────────────────────────────────────┐
│  의류         장난감        유아용품      교육     │

## 4. 화면 흐름도 (Screen Flow)

### 4.1 상품 구매 플로우

```
[홈/카테고리/검색]
        │
        ▼
  [상품 목록] ◀──── [필터/정렬 변경]
        │
        ▼
  [상품 상세]
    │       │
    ▼       ▼
[장바구니]  [바로구매]
    │           │
    ▼           │
[장바구니 페이지]│
    │           │
    └─────┬─────┘
          ▼
    [로그인 확인] ── 미로그인 ──▶ [로그인] ──▶ [콜백으로 복귀]
          │ 로그인됨
          ▼
    [주문서 작성]
      │       │         │
      ▼       ▼         ▼
  [배송지 선택] [쿠폰 적용] [포인트 사용]
      │
      ▼
  [결제 진행 (PG)]
      │           │
      ▼           ▼
  [결제 완료]  [결제 실패]
      │           │
      ▼           ▼
  [주문 상세]  [재시도/장바구니]
```

### 4.2 회원가입/로그인 플로우

```
[서비스 접근]
      │
      ▼
  [인증 필요?] ── No ──▶ [서비스 이용]
      │ Yes
      ▼
  [로그인 페이지]
    │              │
    ▼              ▼
[이메일 로그인]  [소셜 로그인 (카카오/네이버/구글)]
    │              │
    ▼              ▼
[인증 성공?]    [OAuth 콜백]
    │ Yes           │
    │              ▼
    │         [기존 회원?] ── Yes ──▶ [로그인 완료]
    │              │ No
    │              ▼
    │         [추가 정보 입력 (닉네임, 휴대폰, 약관)]
    │              │
    ▼              ▼
[로그인 완료] ◀──────┘
      │
      ▼
[callbackUrl 또는 홈]
      │
      ▼ (선택)
[자녀 정보 입력]
```

### 4.3 커뮤니티 글쓰기 플로우

```
[커뮤니티] → [게시판 선택] → [글쓰기 클릭]
                                  │
                                  ▼
                          [로그인 확인] ── 미로그인 ──▶ [로그인]
                                  │ 로그인됨
                                  ▼
                          [글 작성 에디터]
                            │ 게시판 선택
                            │ 제목 (2~100자)
                            │ 본문 (Rich Text, 이미지, 최대 10,000자)
                            │ 태그 (최대 5개)
                            │ 연관 상품 (제품후기만)
                                  │
                                  ▼
                          [금칙어 검증] ── 실패 ──▶ [경고 표시]
                                  │ 통과
                                  ▼
                          [게시] ──▶ [게시글 상세]
                                     + 포인트 적립 알림
```

### 4.4 주문 취소/반품 플로우

```
[마이페이지 > 주문 내역]
        │
        ▼

        │

        │                                    │
        │                                    ▼
        │                            [취소 사유 선택]
        │                                    │
        │                                    ▼

        │
        └── 배송 완료 후 7일 이내 ──▶ [반품/교환 신청]
                                         │
                                         ▼
                                   [사유 선택 + 수거 신청]
                                         │
                                         ▼
                                   [수거 → 검수 → 환불/교환]
```

## 5. 콘텐츠 계층 구조

### 5.1 상품 정보 계층

```
상품 (Product)
├── 기본 정보
│   ├── 상품명, 슬러그
│   ├── 가격 (정가, 할인가, 할인율)
│   ├── 브랜드
│   ├── 카테고리 (대 > 중 > 소)
│   ├── 적정 연령대
│   └── 태그
├── 미디어
│   ├── 대표 이미지 (thumbnail)
│   └── 상세 이미지 갤러리 (최대 10장)
├── 옵션 (ProductOption[])
│   ├── 옵션명 (색상/사이즈 등)
│   ├── 추가 금액
│   ├── 개별 재고
│   └── SKU 코드
├── 구매 정보
│   ├── 총 재고 / 판매 수량
│   ├── 배송비 정책
│   ├── 최대 주문 수량
│   └── 반품/교환 정책
├── 상세 콘텐츠
│   ├── 상품 설명 (Rich Text HTML)
│   └── 상세 스펙 테이블
└── 사용자 생성 콘텐츠 (UGC)
    ├── 리뷰 (평점, 텍스트, 이미지)
    │   ├── 평균 평점 / 리뷰 수
    │   └── 평점 분포 차트
    └── Q&A
```

### 5.2 커뮤니티 콘텐츠 계층

```
게시글 (Post)
├── 메타 정보
│   ├── 게시판 카테고리 (boardSlug)
│   ├── 작성자 (닉네임, 프로필이미지)
