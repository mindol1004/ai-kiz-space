# 키즈공간 기능 명세서

## 1. 인증 시스템

### FN-001: 회원가입

| 항목 | 내용 |
|------|------|
| **기능 ID** | FN-001 |
| **우선순위** | P0 |
| **관련 화면** | SCR-AUTH-001, SCR-AUTH-002 |

#### 상세 설명

사용자가 이메일 또는 소셜 계정(카카오, 네이버, 구글)을 통해 서비스에 가입할 수 있다.

#### 이메일 회원가입 입력/출력

| 항목 | 타입 | 필수 | 유효성 검증 | 설명 |
|------|------|------|-------------|------|
| email | string | O | 이메일 형식, 중복 확인 | 로그인 ID |
| password | string | O | 8~20자, 영문+숫자+특수문자 포함 | 비밀번호 |
| passwordConfirm | string | O | password와 일치 | 비밀번호 확인 |
| name | string | O | 2~20자, 한글/영문 | 이름 |
| nickname | string | O | 2~15자, 중복 확인 | 닉네임 |
| phone | string | O | 010-XXXX-XXXX 형식 | 휴대폰 번호 |
| agreeTerms | boolean | O | true | 이용약관 동의 |
| agreePrivacy | boolean | O | true | 개인정보처리방침 동의 |
| agreeMarketing | boolean | X | - | 마케팅 수신 동의 |

#### 소셜 회원가입 절차

1. 소셜 로그인 버튼 클릭
2. 소셜 서비스 OAuth 인증
3. 최초 로그인 시 추가 정보 입력 (닉네임, 휴대폰, 약관 동의)
4. 가입 완료

#### 비즈니스 규칙

- BR-001: 동일 이메일로 중복 가입 불가
- BR-002: 소셜 계정 이메일이 기존 이메일 회원과 동일하면 계정 연동 안내
- BR-003: 14세 미만 가입 시 법정대리인 동의 필요 (향후 구현)
- BR-004: 가입 완료 시 환영 쿠폰 자동 발급

#### 예외 처리

- ERR-001: 이미 사용 중인 이메일 → "이미 가입된 이메일입니다."
- ERR-002: 이미 사용 중인 닉네임 → "이미 사용 중인 닉네임입니다."
- ERR-003: 비밀번호 조건 미충족 → 구체적 미충족 조건 표시
- ERR-004: 소셜 인증 실패 → "소셜 로그인에 실패했습니다. 다시 시도해주세요."

### FN-002: 로그인/로그아웃

| 항목 | 내용 |
|------|------|
| **기능 ID** | FN-002 |
| **우선순위** | P0 |
| **관련 화면** | SCR-AUTH-001 |

#### 상세 설명

이메일+비밀번호 또는 소셜 계정으로 로그인하며, 로그인 상태를 유지하거나 로그아웃할 수 있다.

#### 비즈니스 규칙

- BR-005: 로그인 5회 실패 시 5분간 로그인 차단
- BR-006: "로그인 유지" 체크 시 30일간 세션 유지, 미체크 시 브라우저 종료 시 만료
- BR-007: 비밀번호 5회 오류 시 비밀번호 재설정 안내
- BR-008: 관리자 계정은 별도 관리자 로그인 경로 사용

## 2. 사용자 프로필

### FN-003: 프로필 관리

| 항목 | 내용 |
|------|------|
| **기능 ID** | FN-003 |
| **우선순위** | P0 |
| **관련 화면** | SCR-MY-001, SCR-MY-002 |

#### 상세 설명

사용자가 프로필 정보를 조회하고 수정할 수 있다. 프로필 이미지, 닉네임, 자녀 정보 등을 관리한다.

#### 자녀 정보 입력

| 항목 | 타입 | 필수 | 설명 |
|------|------|------|------|
| childName | string | X | 자녀 이름/별명 |
| birthDate | date | O | 생년월일 (연령 자동 계산) |
| gender | enum | X | 남아/여아/선택안함 |

#### 비즈니스 규칙

- BR-009: 자녀 정보는 최대 5명까지 등록 가능
- BR-010: 자녀 연령은 생년월일 기준 자동 계산
- BR-011: 프로필 이미지는 5MB 이하, JPG/PNG/WebP 허용

## 3. 상품 관리

### FN-004: 상품 카탈로그

| 항목 | 내용 |
|------|------|
| **기능 ID** | FN-004 |
| **우선순위** | P0 |
| **관련 화면** | SCR-SHOP-001, SCR-SHOP-002 |

#### 상세 설명

카테고리별로 상품을 조회하고, 다양한 조건으로 필터링/정렬할 수 있다.

#### 필터 옵션

| 필터 | 타입 | 옵션 |
|------|------|------|
| 카테고리 | 다중 선택 | 대분류/중분류/소분류 |
| 가격대 | 범위 | 최소~최대 (슬라이더) |
| 연령대 | 다중 선택 | 0~6개월, 6~12개월, 1~2세, 3~5세, 6~8세, 9세 이상 |
| 브랜드 | 다중 선택 | 브랜드 목록 |
| 배송 | 단일 선택 | 무료배송, 당일배송 |
| 할인 | 체크박스 | 할인 상품만 |

#### 정렬 옵션

| 정렬 | 설명 |
|------|------|
| 추천순 | 기본값, 판매량+리뷰 종합 |
| 신상품순 | 등록일 내림차순 |
| 판매량순 | 판매 수량 내림차순 |
| 가격 낮은순 | 판매가 오름차순 |
| 가격 높은순 | 판매가 내림차순 |
| 리뷰 많은순 | 리뷰 수 내림차순 |
| 평점 높은순 | 평균 평점 내림차순 |

#### 비즈니스 규칙

- BR-012: 페이지당 20개 상품 표시 (무한 스크롤 또는 페이지네이션)
- BR-013: 품절 상품은 목록 하단에 표시, "품절" 배지 노출
- BR-014: 할인 상품은 할인율 배지 표시
- BR-015: 카테고리는 최대 3depth까지 지원

### FN-005: 상품 상세

| 항목 | 내용 |
|------|------|
| **기능 ID** | FN-005 |
| **우선순위** | P0 |
| **관련 화면** | SCR-SHOP-003 |

#### 상세 설명

상품의 상세 정보를 확인하고, 옵션을 선택하여 장바구니에 담거나 바로 구매할 수 있다.

#### 화면 구성 요소

| 영역 | 내용 |
|------|------|
| 이미지 갤러리 | 상품 이미지 슬라이더 (최대 10장), 확대 보기 |
| 기본 정보 | 브랜드, 상품명, 가격, 할인 정보, 적정 연령 |
| 옵션 선택 | 색상, 사이즈 등 옵션 선택기, 수량 조절 |
| 액션 버튼 | 장바구니 담기, 바로 구매, 위시리스트 |
| 상세 탭 | 상품 설명, 상세 스펙, 배송/교환/반품, 리뷰, Q&A |
| 추천 상품 | 함께 본 상품, 같은 카테고리 인기 상품 |

#### 비즈니스 규칙

- BR-016: 품절 옵션은 선택 불가 처리
- BR-017: 최소 주문 수량 1개, 최대 주문 수량 99개 (상품별 설정 가능)
- BR-018: 상품 상세 이미지는 지연 로딩(lazy loading) 적용
- BR-019: 가격 변동 시 장바구니에 담긴 상품도 자동 반영

### FN-006: 상품 검색

| 항목 | 내용 |
|------|------|
| **기능 ID** | FN-006 |
| **우선순위** | P0 |
| **관련 화면** | SCR-SHOP-004 |

#### 상세 설명

키워드를 입력하여 상품을 검색하고, 검색 결과를 필터링/정렬할 수 있다.

#### 검색 기능

| 기능 | 설명 |
|------|------|
| 키워드 검색 | 상품명, 브랜드명, 카테고리, 태그에서 검색 |
| 자동완성 | 입력 중 추천 검색어 표시 (디바운스 300ms) |
| 최근 검색어 | 최근 10개 검색어 로컬 저장, 삭제 가능 |
| 인기 검색어 | 실시간 인기 검색어 Top 10 표시 |
| 검색 결과 없음 | 유사 상품/인기 상품 추천 |

#### 비즈니스 규칙

- BR-020: 검색어 최소 1자 이상
- BR-021: 특수문자, SQL 인젝션 필터링
- BR-022: 검색 결과는 상품 목록과 동일한 필터/정렬 지원

## 4. 장바구니 & 주문

### FN-007: 장바구니

| 항목 | 내용 |
|------|------|
| **기능 ID** | FN-007 |
| **우선순위** | P0 |
| **관련 화면** | SCR-CART-001 |

#### 상세 설명

선택한 상품을 장바구니에 담고, 수량 변경/삭제 후 주문으로 진행할 수 있다.

#### 기능 목록

| 기능 | 설명 |
|------|------|
| 상품 추가 | 상품 상세에서 옵션/수량 선택 후 장바구니 담기 |
| 수량 변경 | + / - 버튼 또는 직접 입력 |
| 상품 삭제 | 개별 삭제, 선택 삭제 |
| 전체 선택/해제 | 체크박스로 주문할 상품 선택 |
| 가격 계산 | 상품 금액, 할인 금액, 배송비, 총 결제 금액 표시 |
| 옵션 변경 | 장바구니에서 옵션 변경 가능 |

#### 비즈니스 규칙

- BR-023: 비회원도 장바구니 이용 가능 (로컬 스토리지), 로그인 시 서버와 병합
- BR-024: 장바구니 최대 50개 상품
- BR-025: 품절 상품은 장바구니에서 "품절" 표시, 주문 불가
- BR-026: 배송비: 30,000원 이상 무료, 미만 시 3,000원

### FN-008: 주문/결제

| 항목 | 내용 |
|------|------|
| **기능 ID** | FN-008 |
| **우선순위** | P0 |
| **관련 화면** | SCR-ORDER-001, SCR-ORDER-002, SCR-ORDER-003 |

#### 상세 설명

장바구니에서 선택한 상품 또는 바로 구매 상품에 대해 배송 정보를 입력하고 결제를 진행한다.

#### 주문서 작성 항목

| 항목 | 타입 | 필수 | 설명 |
|------|------|------|------|
| 배송지 선택 | select | O | 기존 배송지 선택 또는 신규 입력 |
| 수령인 | string | O | 수령인 이름 |
| 연락처 | string | O | 수령인 연락처 |
| 주소 | string | O | 우편번호 + 기본주소 + 상세주소 |
| 배송 메모 | string | X | 배송 요청사항 (직접입력 또는 선택) |
| 쿠폰 적용 | select | X | 사용 가능 쿠폰 목록에서 선택 |
| 포인트 사용 | number | X | 보유 포인트 내에서 사용 |
| 결제 수단 | select | O | 카드, 무통장, 간편결제 등 |

#### 결제 수단

| 수단 | 설명 |
|------|------|
| 신용/체크카드 | PG사 연동 카드 결제 |
| 무통장입금 | 가상계좌 발급 (24시간 이내 입금) |
| 카카오페이 | 카카오페이 간편결제 |
| 네이버페이 | 네이버페이 간편결제 |
| 토스페이 | 토스페이 간편결제 |

#### 비즈니스 규칙

- BR-027: 주문 시 재고 실시간 확인, 재고 부족 시 주문 불가
- BR-028: 무통장입금 24시간 미입금 시 자동 취소
- BR-029: 포인트는 1,000포인트 이상부터 100포인트 단위로 사용
- BR-030: 결제 완료 시 주문 확인 이메일/SMS 발송
- BR-031: 결제 실패 시 사유 안내 및 재시도 유도

### FN-009: 주문 관리

| 항목 | 내용 |
|------|------|
| **기능 ID** | FN-009 |
| **우선순위** | P0 |
| **관련 화면** | SCR-MY-003, SCR-MY-004 |

#### 주문 상태 흐름

```
[주문접수] → [결제완료] → [상품준비중] → [배송중] → [배송완료]
                ↓              ↓            ↓
          [주문취소]    [주문취소]     [반품/교환 신청]
                                          ↓
                                   [반품/교환 처리중]
                                          ↓
                                   [반품/교환 완료]
```

#### 기능 목록

| 기능 | 조건 | 설명 |
|------|------|------|
| 주문 목록 조회 | - | 기간별 필터, 상태별 필터 |
| 주문 상세 조회 | - | 주문 정보, 상품 정보, 배송 정보, 결제 정보 |
| 주문 취소 | 배송 전 | 결제 완료~상품준비중 단계에서 가능 |
| 반품 신청 | 배송 완료 후 7일 이내 | 반품 사유 선택, 수거 신청 |
| 교환 신청 | 배송 완료 후 7일 이내 | 교환 사유, 교환 옵션 선택 |
| 배송 추적 | 배송중 | 택배사 배송 조회 연동 |
| 구매 확정 | 배송 완료 후 | 수동 확정 또는 7일 후 자동 확정 |

## 5. 상품 리뷰

### FN-010: 상품 리뷰 시스템

| 항목 | 내용 |
|------|------|
| **기능 ID** | FN-010 |
| **우선순위** | P1 |
| **관련 화면** | SCR-SHOP-003 (상품 상세 내 리뷰 탭) |

#### 리뷰 작성

| 항목 | 타입 | 필수 | 설명 |
|------|------|------|------|
| rating | number (1~5) | O | 별점 |

- BR-035: 욕설/비방/광고 포함 리뷰는 관리자 검토 후 비노출 처리
- BR-036: 리뷰 정렬: 최신순, 평점 높은순, 평점 낮은순, 포토 리뷰만
