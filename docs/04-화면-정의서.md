# 키즈공간 화면 정의서

## 1. 공통 컴포넌트

### 1.1 헤더 (Header)

| ID | SCR-COMMON-001 |
|----|----------------|
| **위치** | 모든 페이지 상단 고정 |
| **컴포넌트 타입** | Server Component (일부 Client) |

#### 데스크톱 레이아웃

```
┌─────────────────────────────────────────────────────────────────┐
│ [로고]   [쇼핑 ▾]  [커뮤니티]  [기획전]                          │
│                          [🔍 검색바]  [🛒 장바구니(n)] [👤 로그인] │
└─────────────────────────────────────────────────────────────────┘
```

#### 모바일 레이아웃

```
┌──────────────────────────────────┐
│ [☰ 메뉴]  [로고]  [🔍] [🛒(n)] │
└──────────────────────────────────┘
```

#### 상태별 표시

| 상태 | 표시 내용 |
|------|-----------|
| 비로그인 | [로그인] [회원가입] |
| 로그인 | [프로필 아이콘 ▾] → 드롭다운: 마이페이지, 주문내역, 로그아웃 |
| 장바구니 | 아이콘 + 뱃지(담긴 수량), 0이면 뱃지 미표시 |
| 관리자 | [관리자] 버튼 추가 표시 |
