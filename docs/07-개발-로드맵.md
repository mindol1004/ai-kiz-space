# 키즈공간 개발 로드맵

## 1. 프로젝트 일정 개요

### 전체 타임라인

```
Phase 1 (MVP)       Phase 2 (확장)       Phase 3 (고도화)
[───── 8주 ─────]  [───── 6주 ─────]  [───── 6주 ─────]
 Sprint 1~4          Sprint 5~7          Sprint 8~10
```

| Phase | 기간 | 목표 |
|-------|------|------|
| **Phase 1 (MVP)** | 8주 (Sprint 1~4) | 핵심 쇼핑 + 커뮤니티 기능, 서비스 런칭 가능 수준 |
| **Phase 2 (확장)** | 6주 (Sprint 5~7) | UX 고도화, 마케팅 기능, 운영 도구 강화 |
| **Phase 3 (고도화)** | 6주 (Sprint 8~10) | 개인화, 성능 최적화, 사업 확장 기능 |

## 6. 기술 의존성 맵

```
Sprint 1 (기반)
├── 인증 시스템 ──────────────────────────────────────┐
├── DB 스키마 ────────────────────────────────────────┤
├── 공통 UI ─────────────────────────────────────────┤
└── API 유틸 ─────────────────────────────────────────┤
                                                      ▼
Sprint 2 (쇼핑)                                    Sprint 3 (주문/커뮤니티)
├── 상품 API ──────────────────────────────────────▶ 주문 API
├── 카테고리 ──────────────────────────────────────▶ 상품 필터
├── 장바구니 ──────────────────────────────────────▶ 주문서
└── 검색 ─────────────────────────────────────────▶ 커뮤니티
                                                      ▼
                                                   Sprint 4 (마이페이지/관리자)
                                                   ├── 마이페이지 (주문, 게시글 의존)
                                                   └── 관리자 (전체 도메인 의존)
```
