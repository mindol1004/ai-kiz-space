# 프로모션 기획 (Promotion Planning)

## 한 줄 요약
쇼핑몰의 판매 촉진을 위한 다양한 프로모션 전략을 기획하고 실행하는 스킬

## 상세 설명
프로모션 기획은 단기적으로 매출을 증대하고 고객을 획득/유지하기 위한 전략적 역량이다. 이 스킬은 프로모션 유형 선택(할인, 쿠폰, 사은품, 경품, 타임딜 등), 예산 수립과 ROI 예측, 프로모션 일정 설계, 참여 조건 및 규칙 제정, 성과 측정과 학습 체계 구축을 포함한다. 브랜드 가치와의 일관성 유지, 마진 관리, 고객 품질(할인만을 위한 고객 vs 충성 고객) 관리 등 장기적 관점에서의 프로모션 설계가 핵심이다.

## 사용 시나리오 예시

**연간 프로모션 캘린더 수립**
올해 4대 시즌(설날/추석, 창립기념일, 11.11, 연말)의 프로모션 계획을 수립한다. 각 시즌의 핵심 메시지, 할인율 범위, 대상 상품군, 예상 매출 목표를 설정하고, 내부 역량(재고, 시스템, 고객센터)과 외부 환경(경쟁사 동향)을 고려하여 일정을 확정한다. 특히 11.11은 연간 매출의 15%를 차지하는 핵심 이벤트로서, 9월부터 상품 선정, 재고 확보, 마케팅 준비를 본격화한다.

**새로운 프로모션 유형 도입**
기존 단순 할인 프로모션에 지친 고객을 위해 새로운 프로모션 유형을 도입한다. "회원 한정 선착순 판매", "구매 금액에 따른 레벨업 보상", "친구 초대 시 상호 혜택" 등 참여 유도 요소가 있는 프로모션을 설계하고, A/B 테스트로 효과를 검증한다.

## 산출물/템플릿

| 산출물 | 내용 |
|--------|------|
| 연간 프로모션 캘린더 | 12개월간 주요 프로모션 일정과 내용을 한눈에 |
| 개별 프로모션 기획서 | 목표, 대상, 혜택, 조건, 일정, 예상 효과를 포함한 상세 기획서 |
| 프로모션 성과 리포트 | 프로모션별 매출, 참여율, ROI, 교훈 정리 |
| 프로모션 가이드라인 | 프로모션 유형별 운영 원칙과 주의사항 |

**프로모션 기획서 템플릿**

```
┌─────────────────────────────────────────────────────────────────────┐
│ [프로모션명] 기획서                                                 │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│ 1. 프로모션 개요                                                   │
│ • 기간: 2025.11.1 ~ 11.11                                          │
│ • 유형: 전매장 할인 + 쿠폰                                         │
│ • 목표: 매출 30% 증가, 신규 고객 2만 명 획득                       │
│                                                                     │
│ 2. 프로모션 상세                                                   │
│ • 할인 내용:                                                       │
│ - 3만원 이상 구매 시 5천원 할인 (상한 없음)                        │
│ - 회원 전용: 추가 10% 할인                                         │
│ - 신규 회원: 첫 구매 2만원 즉시 할인                               │
│ • 적용 대상: 전 카테고리 (일부 제외)                               │
│ • 사용 조건: 1회 주문 10만원 이상 시 사용 가능                     │
│                                                                     │
│ 3. 예상 효과 및 리스크                                             │
│ • 매출 기여: 15억원 (전년 동기 대비 25%)                           │
│ • 마진율: 예상 32% (평소 38%)                                      │
│ • 리스크: 대규모 할인으로 인한 수익 감소, 경쟁사 대응              │
│ • 대응: 고객단가 상품 별도 할인, 회원 차별화                       │
│                                                                     │
│ 4. 운영 계획                                                       │
│ • 상품 비축: 10월 말 기준 재고 150% 확보                           │
│ • 시스템: 11/1 부하 테스트 완료                                    │
│ • 고객센터: 임시 상담원 10명 증원                                  │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```
