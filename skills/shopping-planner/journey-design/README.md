# ${skill} Skill

## 역할
이 스킬은 쇼핑몰 기획 프로세스의 특정 단계를 담당합니다.

## 파라미터
스킬별 필요한 파라미터를 `parameters` 객체로 전달합니다.

## 의존성
선행 실행되어야 하는 스킬들이 있습니다 (스킬 레지스트리 참고).

## 사용
에이전트가 프롬프트를 받으면 자동으로 필요한 스킬이 선택되고 실행됩니다.

## 구현 상세
실제 구현 코드는 core/main-agent.md의 "스킬 구현 템플릿"을 참고하세요.

## 출력
`{ status: 'success'|'error', result: any, summary: string }`
