# 마케팅 채널 전략 (Marketing Channel Strategy)

## 한 줄 요약
다양한 마케팅 채널의 특성을 이해하고, 최적의 채널 조합과 투자 배분을 결정하는 스킬

## 상세 설명
마케팅 채널 전략은 각 채널의 특성과 효율성을 이해하여 제한된 예산을 최적화하는 역량이다. 이 스킬은 채널별 특성과 역할 정의(인지/구매/유지), 채널별 ROI 분석, 옴니채널 통합 전략, 신규 채널 탐색과 활용, 채널 간 시너지 극대화 등을 포함한다. 검색 마케팅(SEO/SEM), 소셜 미디어 마케팅, 콘텐츠 마케팅, 이메일 마케팅, 인플루언서 마케팅, 제휴 마케팅, 오프라인 채널 등 다양한 채널의 특성과 최신 트렌드를 파악하고, 비즈니스 단계와 목표에 맞는 채널 포트폴리오를 설계하는 능력이 핵심이다.

## 사용 시나리오 예시

**채널 투자 최적화**
올해 마케팅 예산 10억 원의 배분 계획을 수립한다. 작년 각 채널별 ROI 데이터 분석 결과: 검색광고 ROI 3.2x, 소셜광고 2.8x, 이메일 5.5x, 인플루언서 2.1x. 이를 바탕으로 고 ROI 채널(이메일, 검색) 투자를 늘리고, 저 ROI 채널은 최적화하거나 축소한다. 그리고 틱톡에서 신규 채널 테스트를 통해 신규 고객층을 탐색한다.

**신규 채널 진입**
20대 타겟을 위해 틱톡 마케팅을 도입하고자 한다. 경쟁사의 틱톡 계정 분석, 타겟 고객의 콘텐츠 소비 패턴, 틱톡 광고 플랫폼 특성과 비용 구조를 조사한다. 크리에이터 협업과 오가닉 콘텐츠의 최적 조합을 연구하고, 6개월 진입 계획을 수립한다.

## 산출물/템플릿

| 산출물 | 내용 |
|--------|------|
| 채널별 ROI 분석표 | 채널별 투자 비용, 매출, ROI, CAC 비교 분석 |
| 채널 포트폴리오 전략서 | 채널별 역할, 투자 비중, 목표 설정 |
| 연간 마케팅 캘린더 | 채널별 캠페인 일정과 연동 계획 |
| 채널별 가이드라인 | 각 채널의 운영 원칙과 베스트 프랙티스 |

**채널 투자 배분 최적화 모델**

```
현재 배분                    최적화 후 배분
┌──────────────┐             ┌──────────────┐
│ 검색 광고 40% │             │ 검색 광고 35% │
│ SNS 광고 30% │      →      │ 이메일 25%   │
│ 인플루언서 15%│             │ SNS 광고 20% │
│ 이메일 5%    │             │ 콘텐츠 마케팅 10% │
│ 기타 10%     │             │ 인플루언서 5% │
└──────────────┘             │ 테스트 예산 5% │
                             └──────────────┘

예상 효과:
- 총 마케팅 비용: 동일
- 예상 매출 증가: +18%
- 전반적 ROI: 2.8x → 3.5x
