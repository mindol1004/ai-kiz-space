# 바이럴 마케팅 설계 (Viral Marketing Design)

## 한 줄 요약
고객의 자발적 공유와 확산을 유도하여 비용 효율적인 고객 획득과 브랜드 인지도 구축을 달성하는 스킬

## 바이럴 마케팅 설계는 고객이 자발적으로 브랜드를 친구, 가족, 지인에게 추천하도록 유도하는 전략적 역량이다. 이 스킬은 바이럴 원리 이해(social currency, public visibility, practical value, story), 공유 동기 분석, 인센티브 구조 설계, 확산 경로 최적화, UGC(User Generated Content) 활성화 등을 포함한다. 단순한 "공유하면 할인"을 넘어서 공유하고 싶은 내용이 담긴 콘텐츠, 공유하기 쉬운 메커니즘, 공유에 대한 보상(물질적/정서적)을 설계하는 능력이 핵심이다.

## 사용 시나리오 예시

**친구 초대 프로그램 설계**
기존 고객이 새 고객을 초대하면 양측에 benefit을 제공하는 프로그램을 설계한다. 초대자의 사회적 동기(친구에게 도움이 되고자 하는 욕구)와 물질적 동기(할인 혜택)를 동시에 자극하도록 "친구 초대 시 나도 친구도 5,000원 할인"이라는 메시지를 설계한다. 초대 링크 한 번으로 친구의 첫 구매까지 트래킹하여 인센티브를 지급하고, 초대 성과 상위자에게 추가 보상을 제공하여 경쟁심을 유발한다.

**UGC 캠페인 기획**
고객이 자발적으로 商品使用후기나스타일을 공유하도록 유도하는 캠페인 진행한다. "나의OOTD"라는 해시태그를 통해 고객의 룩북을收集하고, 우수 콘텐츠에 대해 공식 계정에서 repost 및 소정의 상품권을 제공한다. 참여 고객은 노출에 대한 만족감을, 브랜드는 믿을 수 있는 콘텐츠와 자연스러운 확산을 얻는다.

## 산출물/템플릿

| 산출물 | 내용 |
|--------|------|
| 바이럴 캠페인 기획서 | 캠페인 목표, 메커니즘, 인센티브, 확산 경로 설계 |
| 친구 초대 프로그램 설계서 | 초대 로직, 보상 구조, 트래킹 체계 |
| UGC 활성화 계획 | 콘텐츠征集, 선별, 활용, 보상 체계 |
| 바이럴 계수(K-factor) 모니터링 대시보드 | K-factor, 바이럴률, 참여율 추이 |

---

**템플릿 구조 예시: 바이럴 캠페인 설계 프레임워크**

```
┌─────────────────────────────────────────────────────────────────────────┐
│                    바이럴 캠페인 설계 요소                               │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                         │
│  ┌──────────────────┐                                                  │
│  │ 바이럴의 5가지 요소 │                                                │
│  ├──────────────────┤                                                  │
│  │ 1. Social Currency (社交価値)                                      │
│  │    - 공유하면 나도 트렌디해 보임                                    │
│  │    - 독특한 경험,稀缺한 혜택, 커뮤니티 소속감                       │
│  │                                                                  │
│  │ 2. Public Visibility (공공성)                                      │
│  │    - 다른 사람도 볼 수 있음                                         │
│  │    - 뱃지, 랭킹, 공유 가능한 결과물                                 │
│  │                                                                  │
│  │ 3. Practical Value (실용적 가치)                                   │
│  │    - 공유를 통해 친구에게 실질적 도움                               │
│  │    - 유용한 정보,优惠정보, 꿀팁                                    │
│  │                                                                  │
│  │ 4. Story (이야기성)                                                │
│  │    - 단순 정보가 아닌 이야기として 전달                             │
│  │    - "이렇게 사용했어요" 형태의 서사                               │
│  │                                                                  │
│  │ 5. Trigger (트리거)                                                │
│  │    - 공유冲動을 일으키는 자극                                      │
│  │    - 타이밍, 상황, 감정적 순간                                     │
│  └──────────────────┘                                                  │
│                                                                         │
├─────────────────────────────────────────────────────────────────────────┤
│  바이럴 효과 측정: K-Factor                                            │
│                                                                         │
│  K-Factor = (초대당 평균 초대자 수) × (초대 전환율)                     │
│                                                                         │
│  K > 1: 指数级增长 (한 명이 한 명 이상 초대)                           │
│  K = 1: 线性增长 (균형)                                                │
│  K < 1: 衰减增长 (확산 후 소멸)                                        │
│                                                                         │
│  예시:                                                                │
│  - 1,000명의 고객이 각각 2명을 초대                                    │
│  - 초대받은 고객 중 50%가 첫 구매                                      │
│  - K = 2 × 0.5 = 1.0 (균형 성장)                                      │
│                                                                         │
└─────────────────────────────────────────────────────────────────────────┘
```

**친구 초대 프로그램 설계 예시**

| 요소 | 설계 내용 |
|------|-----------|
| 메커니즘 | 고유 초대 링크 발급 (웹/APP) |
| 인센티브 구조 | 초대자: 친구 첫 구매 시 5,000원 / 친구: 첫 구매 5,000원 |
| 추가 인센티브 | 5명 초대 시 추가 10,000원 / 10명 초대 시 VIP 등급 |
| 트래킹 | 초대 링크 클릭 → 세션 유지 → 구매 완료 시 매칭 |
| 한도 | 월 최대 50,000원 상당 (spam 방지) |
| 만료 | 초대 링크 발급 후 30일간 유효 |
| 소셜 연동 | Kakao, SMS, Email, SNS 공유 버튼 |
