# KPI 설계 (KPI Design)

## 한 줄 요약
쇼핑몰의 목표 달성을 측정하기 위한 핵심 성과 지표(KPI)를 체계적으로 설계하고 관리하는 스킬

## 상세 설명
KPI 설계는 비즈니스 목표를 측정 가능한 지표로 전환하고, 조직 전체가 동일한 방향으로 움직이도록 하는 기반 역량이다. 이 스킬은 목표와 전략에 맞는 KPI 선택, 계층적 KPI 구조(전략-운영-실행), 지표 정의 및 계산 방식 표준화, 목표값 산출, 대시보드 설계, KPI 리뷰 프로세스 구축을 포함한다. 단순히 많은 지표를 측정하는 것이 아니라, 의사결정에 실제로 활용되는 핵심 지표를 선별하고, 그 지표가 업무 행동을 어떻게 유도하는지를 고려하는 것이 핵심이다.

## 사용 시나리오 예시

**전사 KPI 체계 수립**
올해 회사 목표(매출 30% 성장, 마진 5% 개선)를 달성하기 위한 KPI 체계를 수립한다. 전략 KPI로 매출 성장률, 마진율, 고객 생애 가치를 설정하고, 각 부서별 운영 KPI를 세분화한다. 마케팅은 신규 고객 CAC와 전환율, 상품부는 재고 회전율과 품절률, CS는 CSAT와 해결률을 각각 담당하도록 하고, 모든 KPI를 한눈에 볼 수 있는 대시보드를 구축한다.

**KPI 리뷰 프로세스 도입**
매주 금요일 1시간 KPI 리뷰 미팅을 도입한다. 전사 KPI와 부서별 KPI의 최근 추이를 확인하고, 목표 대비 이탈이 큰 지표에 대해서는 원인 분석과 대응 방안을 논의한다. KPI 리뷰 결과는 짧은 리포트에 정리하여 경영진에게 보고하고, 주요 변화가 있을 때 즉시 공유하는 체계를 구축한다.

## 산출물/템플릿

| 산출물 | 내용 |
|--------|------|
| KPI 트리 | 전략 KPI에서 세부 운영 KPI로 이어지는 계층 구조도 |
| KPI 정의서 | 각 KPI의 정의, 계산식, 데이터 소스, 목표값, 측정 주기 |
| KPI 대시보드 설계서 | 대시보드 레이아웃, 시각화 방식, 갱신 주기 |
| KPI 리포트 | 주간/월간 KPI 현황 및 분석 |

**KPI 정의서 템플릿**

| 항목 | 내용 |
|------|------|
| KPI명 | 상품 페이지 전환율 |
| 정의 | 상품 페이지 방문 사용자 중 주문 구매한 사용자 비율 |
| 계산식 | (주문 사용자 수 / 상품 페이지 방문 사용자 수) × 100% |
| 데이터 소스 | 웹/앱 분석 시스템, 주문 시스템 |
| 측정 주기 | 일간/주간 |
| 현재 값 | 4.2% |
| 목표값 | 5.0% |
| 담당자 | 운영팀 - 사용자 경험 담당자 |
| 연결 KPI | 전체 GMV, 객단가 |
| 조치 기준 | 4% 미달 시 원인 분석 및 개선 미팅 |
