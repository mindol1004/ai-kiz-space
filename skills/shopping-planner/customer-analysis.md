# 고객 분석 (Customer Analysis)

## 한 줄 요약
고객 데이터를 체계적으로 분석하여 고객 이해를 심화하고 의사결정에 활용하는 스킬

## 상세 설명
고객 분석은 쇼핑몰의 가장 중요한 자산인 고객에 대한 깊은 이해를 바탕으로 전략적 의사결정을 지원하는 역량이다. 이 스킬은 고객 세그먼테이션(RFM, 행동 기반), 고객 생애 가치(CLV) 예측, 획득/유지/이탈 분석, 고객 여정 분석, 고객 니즈 인사이트 도출, 그리고 고객 피드백 분석(리뷰, 설문, 고객센터 기록) 등을 포함한다. 정량적 데이터 분석과 정성적 인사이트를 결합하여 "고객이 누구인가, 무엇을 원하는가, 어떻게 행동하는가"에 대한 답을 도출하고, 이를 상품, 마케팅, 서비스 전략에 반영하는 능력이 핵심이다.

## 사용 시나리오 예시

**고객 세그먼테이션 분석**
전체 고객 100만 명을 대상으로 세그먼테이션을 수행한다. RFM 분석을 통해 8개 세그먼트로 분류하고(Champions, Loyal, Potential, At Risk, Hibernating 등), 각 세그먼트의 특성(구매 패턴, 선호 채널, 가격 민감도)을 파악한다. 결과로 Champion 고객에게는 VIP 서비스, At Risk 고객에게는 Win-back 캠페인, Hibernating 고객에게는 복귀 프로모션을 설계한다.

**고객 이탈 원인 분석**
최근 3개월간 월간 활성 고객이 10% 감소하여 원인을 분석한다. 이탈 고객과 유지 고객의 행동 패턴을 비교한 결과, 이탈 고객은 최근 상품 페이지 조회수 감소, 쿠폰 사용 감소, 고객센터 문의 시 불만 비율이 더 높았다. 세분화된 분석을 통해 "잘못된 할인 코드" 문제가 일부 고객 이탈을 유발했음을 확인하고 즉시 개선 조치를 시행한다.

## 산출물/템플릿

| 산출물 | 내용 |
|--------|------|
| 고객 세그먼트 프로파일 | 각 세그먼트의 특성, 규모, 가치 요약 |
| RFM 분석 리포트 | Recency, Frequency, Monetary별 분포와 세그먼트 분류 |
| CLV 분석 보고서 | 고객 가치 계층화 및 기대 생애 가치 예측 |
| 고객 여정 인사이트 | 세그먼트별 여정 특성과 개선 기회 |
| 이탈 분석 리포트 | 이탈 요인별 분포와 대응 전략 |

**고객 여정 인사이트 분석**

| 여정 단계 | 핵심 지표 | 인사이트 | 개선 과제 |
|-----------|-----------|----------|-----------|
| 인지 | 광고 노출량, 클릭률 | 20대 여성 광고 반응률 높음 | 20대 광고 투자 확대 |
| 고려 | 상품 페이지 체류시간, 장바구니 담기율 | 가격 비교 행동 많음, 의사결정 기간 김 | 사회적 증거 추가 |
| 구매 | 장바구니 포기율, 결제 성공률 | 결제 페이지 포기율 25%, 모바일 결제 어려움 | 결제 절차 간소화 |
| 배송 | 배송 시간 만족도 | 3일 내 배송 만족도 78% | 배송 시효 향상 |
| 사용 | 상품 평가, NPS | 사용 후 평가율 높음, 반품률도 높음 | 정확한 상품 설명 |
| 재구매 | 90일 내 재구매율 | 기존 고객 재구매율 높음(45%), 신규 낮음(15%) | 신규 고객 육성 강화 |
