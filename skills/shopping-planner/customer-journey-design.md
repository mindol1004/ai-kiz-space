# 고객 여정 설계 (Customer Journey Design)

## 한 줄 요약
고객이 쇼핑몰과 상호작용하는 전 과정을 분석하고 최적의 경험을 설계하는 스킬

## 상세 설명
고객 여정 설계는 고객이 브랜드를 처음 접하여 구매, 그리고 리피트까지의 전 과정을 체계적으로 이해하고 최적화하는 역량이다. 인지, 고려, 결정, 구매, 구매 후의 각 단계에서 고객이 무엇을 느끼고, 무엇을 기대하고, 어떤 마찰을 경험하는지를 매핑한다. 이 스킬은 여정 매핑, 페인 포인트/게인 포인트 분석, 접점 최적화, 결정적 순간(Moment of Truth) 관리, 전 채널 통합 등을 포함한다. 정량적 데이터(앱/웹 분석, 전환율)와 정성적 데이터(고객 인터뷰, 리뷰)를 결합하여 고객 관점에서 경험을 개선하는 능력이 핵심이다.

## 사용 시나리오 예시

**여정 최적화 프로젝트**
장바구니에서 구매까지의 전환율이 30%에 그치는 문제를 해결하기 위해 고객 여정을 분석한다. 상품 페이지 탐색 → 장바구니 담기 → 결제 페이지 이동 → 결제 완료 각 단계의 이탈률을 측정한다. 결제 페이지에서 결제 수단 선택 복잡성이 페인 포인트로 식별되어 간소화를 진행하여 전환율을 45%로 향상시킨다.

**신규 고객 온보딩 설계**
신규 고객이 첫 구매까지 평균 12일이 소요되어 최적화를 진행한다. 가입 직후 1시간 내에 웰컴 메시지와 함께 추천 상품을 노출하고, 24시간 내 할인 쿠폰을 제공하여 행동을 유도한다. 첫 구매 완료 시 축하 메시지와 다음 구매용 할인 코드를 제공하여 리피트 전환을 유도한다.

## 산출물/템플릿

| 산출물 | 내용 |
|--------|------|
| 고객 여정 맵 | 각 단계별 행동, 감정, 접점, 페인/게인을 시각화 |
| 접점 최적화 계획 | 각 접촉점별 개선 과제와 우선순위 |
| 결정적 순간 정의서 | 구매 결정에 핵심적인 순간의 관리 지침 |
| 여정 성과 대시보드 | 단계별 전환율, 소요시간, 이탈률 모니터링 |

**템플릿 구조 예시: 고객 여정 맵**

```
┌─────────────────────────────────────────────────────────────────────────┐
│ 고객 여정: 첫 구매 프로세스                                            │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                         │
│ 단계    │ 인지도        │ 고려          │ 결정          │ 구매         │
│         │ (Awareness)   │ (Consider)    │ (Decision)    │ (Purchase)   │
│ ──────────┼─────────────┼──────────────┼──────────────┼──────────────│
│ 행동    │ SNS/검색      │ 상품 비교     │ 리뷰 확인     │ 결제 완료    │
│         │ 광고 클릭     │ 가격 확인     │ 질문 작성     │ 배송 확인    │
│ ──────────┼─────────────┼──────────────┼──────────────┼──────────────│
│ 접점    │ 인스타그램    │ 웹사이트/APP  │ 상품 상세 페이지│ 결제 페이지  │
│         │ 포털사이트    │ 검색 결과     │ 고객센터 문의 │ 주문 페이지  │
│ ──────────┼─────────────┼──────────────┼──────────────┼──────────────│
│ 페인 포인트│ 정보 과다    │ 가격 비교 어려움│ 리뷰 신뢰도 낮음│ 결제 절차 복잡│
│         │ 선택 장애     │ 사이즈 선택 어려움│ 물류 정보 불명확│ 결제 수단 제한│
│ ──────────┼─────────────┼──────────────┼──────────────┼──────────────│
│ 게인 포인트│ 관심 추천    │ 상세 설명     │ 실제 평가     │ 빠른 결제    │
│         │ 신뢰 브랜드    │ 저가 보장     │ 전문 고객센터 │ 할인 코드 사용│
│ ──────────┼─────────────┼──────────────┼──────────────┼──────────────│
│ 감정    │ ───😕──────│ ───🤔───────│ ───🤨──────│ ───😊─────  │
│                                                                         │
│ 전환율  │ → 60%         │ → 45%         │ → 65%         │ → 70%       │
│                                                                         │
└─────────────────────────────────────────────────────────────────────────┘
│
▼
최적화 우선순위:
1. 결제 절차 간소화 (영향 높음, 노력 낮음)
2. 리뷰 시스템 강화 (영향 중간, 노력 중간)
3. 사이즈 추천 도우미 (영향 중간, 노력 높음)
```

**접점별 역할 정의**

| 접점 | 역할 | 핵심 지표 | 최적화 방향 |
|------|------|-----------|-------------|
| 첫 페이지 | 첫인상, 상품 발견 | 이탈률, 체류 시간 | 추천 알고리즘, 시각 디자인 |
| 상품 페이지 | 정보 제공, 구매 충동 | 장바구니 담기율, 전환율 | 콘텐츠 품질, 신뢰 요소 |
| 장바구니 | 최종 의사결정 | 결제 이동률 | 절차 간소화, 인센티브 |
| 결제 페이지 | 거래 완료 | 결제 성공률 | 결제 수단, 신뢰 마크 |
| 주문 페이지 | 기대 관리 | 확인 시간 | 정보 명확, 배송 기대 |
| 물류 추적 | 기대 관리 | 조회 횟수, 만족도 | 실시간 업데이트, 능동 알림 |
