# 고객 여정 설계 (Customer Journey Design)

## 한 줄 요약
고객이 쇼핑몰과 상호작용하는 전 과정을 분석하고 최적의 경험을 설계하는 스킬

## 상세 설명
고객 여정 설계는 고객이 브랜드를 처음 접してから 구매, 그리고 리피트까지의全과정을 체계적으로 이해하고 최적화하는 역량이다. Awareness, Consideration, Decision, Purchase, Post-purchase의 각 단계에서 고객이 무엇을 느끼고, 무엇을期望하고, 어떤 마찰을 경험하는지를 맵핑한다. 이 스킬은 여정 맵핑, Pain Point/Gain Point 분석, 触点 최적화,Moment of Truth 관리, 全渠道整合 등을 포함한다. 정량적 데이터(앱/웹 분석, 전환율)와 정성적 데이터(고객 인터뷰, 리뷰)를 결합하여 고객 관점에서 경험을 개선하는 능력이 핵심이다.

## 사용 시나리오 예시

**여정 최적화 프로젝트**
장바구니에서 구매까지의 전환율이 30%에 그치는 문제를 해결하기 위해 고객 여정을 분석한다. 상품 페이지 浏览-> 장바구니 담기 -> 결제 페이지 이동 -> 결제 완료 각 단계의 이탈률을 측정한다. 결제 페이지에서 결제 수단 선택 복잡성이 Pain Point로 식별되어 Simplification을 진행하여 전환율을 45%로 향상시킨다.

**신규 고객 온보딩 설계**
신규 고객이 첫 구매까지平均 12일이 소요되어 최적화를 진행한다. 加入直후 1시간 内에 웰컴 메시지와 함께 추천 商品을 노출하고, 24시간内有 discounts를 제공하여 행동 유도한다. 첫 구매 완료 시 축하 메시지와 次回使用优惠를 제공하여 리피트 전환을 유도한다.

## 산출물/템플릿

| 산출물 | 내용 |
|--------|------|
| 고객 여정 맵 | 각 단계별 행°, 감정, Touchpoint, Pain/Gain을 시각화 |
| Touchpoint 최적화 계획 | 각 접촉점별 개선 과제와 우선순위 |
| Moment of Truth 정의서 | 구매 결정에 핵심적인 순간의 관리 지침 |
| 여정 성과 대시보드 | 단계별 전환율, 소요시간, 이탈률 모니터링 |

**Touchpoint별 역할 정의**

| Touchpoint | 역할 | 핵심 지표 | 최적화 방향 |
|------------|------|-----------|-------------|
| 首页 | 第一印象, 商品発見 | 跳出率, 停留时间 | 推荐算法, 视觉设计 |
| 商品页 | 情報提供, 购买冲动 | 加购率, 转化率 | 内容质量, 信任元素 |
| 购物车 | 最终决策 | 结账率 | 简化流程, 激励机制 |
| 支付页 | 完成交易 | 支付成功率 | 支付方式, 信任标志 |
| 订单页 | 期待管理 | 确认时间 | 信息清晰, 配送预期 |
| 物流追踪 | 期待管理 | 查询次数, 满意度 | 实时更新, 主动推送 |
