# 개인화 추천 시스템 설계 (Personalization System Design)

## 한 줄 요약
고객의 행동, 선호, 컨텍스트를 분석하여 맞춤형 상품과 콘텐츠를 제공하는 시스템을 설계하는 스킬

## 상세 설명
개인화 추천 시스템은 "올바른 상품, 올바른 고객, 올바른 시점"의 원칙에 따라 각 고객에게 최적의 경험을 제공하는 기술적/전략적 역량이다. 이 스킬은 추천 알고리즘 이해(협업 필터링, 콘텐츠 기반, 하이브리드), 개인화 전략 수립, 데이터 수집 및 처리 파이프라인 설계, A/B 테스트와 피드백 루프 구축을 포함한다. 첫 페이지 개인화 추천, 상품 상세 페이지의 유사 추천, 장바구니 추가 추천, 이메일 마케팅 개인화 등 다양한 접점에서의 개인화 구현, 그리고 개인정보 보호와 투명성 확보가 핵심이다.

## 사용 시나리오 예시

**첫 페이지 추천 시스템 최적화**
첫 페이지 클릭률과 체류 시간이 낮아 개인화 추천으로 개선한다. 사용자 탐색 이력, 구매 이력, 실시간 행동 데이터를 통합하고, 협업 필터링과 콘텐츠 기반 필터링을 결합한 하이브리드 추천 알고리즘을 적용한다. "당신을 위한 추천", "최근 본 상품", "함께 구매한 상품" 등의 모듈을 표시하여 첫날 클릭률 40%, 판매 전환율 25%를 향상시킨다.

**품절 상품 대체 추천**
인기 상품 품절 시 스마트 대체 추천 시스템을 구성한다. 고객이 검색한 상품이 품절일 때, 가격 유사성, 카테고리 근접성, 다른 고객 구매 패턴을 기반으로 가장 적합한 대체 상품을 추천한다. 품절 페이지 전환율이 0%에서 15%로 향상되고 고객 만족도도 높아진다.

## 산출물/템플릿

| 산출물 | 내용 |
|--------|------|
| 개인화 전략 로드맵 | 단기/중기/장기 개인화 구현 계획과 우선순위 |
| 추천 알고리즘 사양서 | 알고리즘 유형, 데이터 소스, 평가 지표 정의 |
| 개인화 A/B 테스트 계획 | 추천 방식별 테스트 설계와 평가 기준 |
| 개인정보 처리 가이드 | 데이터 수집/활용/보관의 투명성 및 규정 준수 가이드 |

**추천 알고리즘 비교 및 적용**

| 알고리즘 | 원리 | 장점 | 한계 | 적용 시나리오 |
|----------|------|------|------|---------------|
| 협업 필터링 | 유사 고객의 행동 기반 | 신규 아이템 추천 가능 | 콜드 스타트 문제 | 리피트 고객에게 |
| 콘텐츠 기반 | 상품 속성 유사도 기반 | 설명 가능성 | 다양성 부족 | 신상품, 데이터 부족시 |
| 딥러닝 | 신경망으로 복잡한 패턴 학습 | 고정밀도, 비선형 관계 포착 | 해석 어려움 | 데이터 풍부한 경우 |
| 지식 기반 | 비즈니스 규칙 기반 | 투명한 설명 | 수동 유지 비용 | 전문 카테고리, 고가품 |
| 하이브리드 | 복수 알고리즘 조합 | 각 알고리즘 보완 | 구현 복잡도 | 대규모 시스템 |

**개인화 효과 측정 지표**

| 지표 | 정의 | 목표 | 측정 주기 |
|------|------|------|----------|
| 추천 클릭률 | 추천 영역 클릭 수 / 추천 영역 노출 수 | >15% | 일간 |
| 추천 전환율 | 추천 경유 구매 수 / 추천 클릭 수 | >5% | 주간 |
| 개인화 커버리지 | 개인화 추천의 주문 비중 | >60% | 주간 |
| 추천 다양성 | 추천 카테고리의 분산도 | 균등분포 | 월간 |
| 사용자 만족도 | 추천 만족도 조사 | >4.0/5.0 | 분기 |
