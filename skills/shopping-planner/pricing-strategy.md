# 가격 전략 수립 (Pricing Strategy)

## 한 줄 요약
비용, 경쟁, 고객 가치 인식을 종합적으로 고려하여 최적의 가격 정책을 설계하는 스킬

## 상세 설명
가격 전략 수립은 상품/서비스의 가치를 적절한 가격으로 표현하여 매출과 이익을 극대화하는 역량이다. 이 스킬은 원가 계산, 가치 기반 가격 설정, 경쟁 가격 분석, 가격 탄력성 분석, 프로모션 가격 설계 등을 포함한다. 마이크로프라이싱(개별 상품 최적가), 심리 가격(9,900원), 번들 가격, 다이내믹 프라이싱(수요 기반 변동가) 등 다양한 가격 설정 기법을 상황에 맞게 적용한다. 또한 가격 변경이 고객 인식, 경쟁 대응, 마진에 미치는 영향을 시뮬레이션하고, 최적 가격 대안을 도출하는 능력이 핵심이다.

## 사용 시나리오 예시

**신규 상품 출시 가격 책정**
신기능이 탑재된 스마트 가전 제품을 출시할 때, 유사 제품 대비 기능 차이를 정량화하고, 고객 설문을 통해 지불의사 수용가를 측정한다. 경쟁사 가격과 자사 원가를 고려하여 마진율이 40% 이상이 되도록 기본 가격을 설정하고, 출시 프로모션으로 초기 인센티브를 제공한다.

**경쟁 대응 가격 전략**
대규모 할인 플랫폼이 동종 상품을 자사보다 30% 저렴하게 판매할 때, 대응 전략을 수립한다. 즉시 가격 매칭, 비가격 경쟁 강화(배송, AS, 사은품), 상품 차별화 등의 옵션을 분석한다. 가격 탄력성 시뮬레이션을 통해 가격 인하 시 매출 영향을 예측하고, 장기 브랜드 가치 훼손을 고려한 최적 대응안을 도출한다.

## 산출물/템플릿

| 산출물 | 내용 |
|--------|------|
| 가격 정책 가이드라인 | 카테고리별 가격대, 마진 목표, 가격 대역폭을 정의한 가이드 |
| 가격 탄력성 분석 리포트 | 상품/카테고리별 가격 변화에 따른 수요 반응 분석 |
| 가격 시뮬레이션 모델 | 가격 변경 시 매출, 이익, 시장점유율 영향 예측 |
| 프로모션 가격 일정표 | 시즌/이벤트별 할인율, 적용 상품, 예상 효과 계획 |

```
┌─────────────────────────────────────────────────────────────┐
│ 가격 책정 결정 요소                                         │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│ 원가 기반      가치 기반       경쟁 기반                    │

│     │              │              │                        │
│     ▼              ▼              ▼                        │
│ 원가 + 마진율   고객 인식 가치  경쟁사 동일                 │
│     └> 최저 가격  └> 최적 가격  └> 전략 가격               │
│                                                             │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│ 가격 책정 기법 선택                                         │
│ ┌─────────────┬─────────────────────────────────────┐      │
│ │ 미끼 가격   │ 동 카테고리 최저가로 유인, 고마진 상품 수익│ │
│ │ 침투 가격   │ 초기 저가로 시장 점유, 이후 점진적 인상   │ │
│ │ 가치 가격   │ 차별화로 고객 가치 제공, 프리미엄 반영    │ │
│ │ 다이내믹 가격│ 수요/재고/시간대별 실시간 조정          │ │
│ │ 번들 가격   │ 조합 판매로 객단가 향상, 재고 정리        │ │
│ │ 심리 가격   │ 9,900원이 아닌 10,000원의 심리 효과      │ │
│ └─────────────┴─────────────────────────────────────┘      │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

| 신상 스킨케어 세트 | 65,000원 | +12% | -9% | 1.0 | 가치 가격, 안정 유지 |
