# 가격 전략 수립 (Pricing Strategy)

## 한 줄 요약
비용, 경쟁, 고객 가치 인식을 종합적으로 고려하여 최적의 가격 정책을 설계하는 스킬

## 상세 설명
가격 전략 수립은 상품/서비스의 가치를 적절한 가격으로 표현하여 매출과 이익을 극대화하는 역량이다. 이 스킬은 原価计算, 価値ベース定价, 競争価格分析, 价格弹性分析, 프로모션 가격 설계 등을 포함한다. 마이크로프레이싱(개별 상품 최적가), 心理定价(9,900원), 捆绑定价(번들), dynamic pricing(수요 기반 변동가) 등 다양한 pricing 기법을 상황에 맞게 적용한다. 또한 가격 변경이 고객 인식, 경쟁 대응, 마진에 미치는 영향을 시뮬레이션하고, 최적 가격 대안을 도출하는 능력이 핵심이다.

## 사용 시나리오 예시

**신규 상품上市 가격 책정**
신기능이 탑재된 스마트 가전 제품을 출시할 때, 유사 제품 대비 기능 차이를 定量化하고, 고객 설문을 통해 지불의사 수용가를 측정한다. 경쟁사 가격과 自社成本을 고려하여 마진율이 40% 이상이 되도록 base price를 설정하고, launch promotion으로 초기 인센티브를 제공한다.

**경쟁 대응 가격 전략**
대규모 할인 플랫폼이 동종 상품을 자사보다 30% 저렴하게 판매할 때, 대응 전략을 수립한다. 즉시降价匹配, 非价格競争强化(배송, AS, 사은품), 商品差异化 등의 옵션을 분석한다. 가격 elasticty 시뮬레이션을 통해降价 시 매출 impact를 예측하고, 장기 브랜드 가치 훼손을 고려한 최적 대응안을 도출한다.

## 산출물/템플릿

| 산출물 | 내용 |
|--------|------|
| 가격 정책 가이드라인 | 카테고리별 가격대, 마진 목표, 가격 대역폭을 정의한 가이드 |
| 가격弹性 분석 리포트 | 상품/카테고리별 가격 변화에 따른 수요 반응 분석 |
| 가격 시뮬레이션 모델 | 가격 변경 시 매출, 이익, 시장점유율 영향 예측 |
| 프로모션 가격 일정표 | 시즌/이벤트별 할인율, 적용 상품, 예상 효과 계획 |

**템플릿 구조 예시: 가격 전략 프레임워크**

```
┌─────────────────────────────────────────────────────────────┐
│                  가격 책정 결정 요소                         │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│   コストベース           バリューベース        競争ベース     │
│     (原価)                 (顧客価値)          (競合価格)     │
│        │                      │                    │        │
│        ▼                      ▼                    ▼        │
│   原価 + 마진율           顧客認識価値          競合 동일     │
│   └> 最低価格             └> 最適価格           └> 戦略価格  │
│                                                             │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│   가격 책정 기법 선택                                        │
│   ┌─────────────┬─────────────────────────────────────┐    │
│   │ 스니킹      │ 同品类最低价引流, 高毛利品赚钱       │    │
│   │ 渗透定价    │ 初期低价抢占市场, 后期逐步提价        │    │
│   │ 价值定价    │ 差异化为客户提供价值, 体现溢价        │    │
│   │ 动态定价    │ 根据需求/库存/时段实时调整            │    │
│   │ 捆绑定价    │ 组合销售提升客单价, 清库存            │    │
│   │ 心理定价    │ 9,900元而非10,000元的心理效应         │    │
│   └─────────────┴─────────────────────────────────────┘    │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

**가격 Elasticity 분석 결과 예시**

| 상품 | 현재가격 | -10%時預測銷售 | +10%時預測銷售 | Elasticity | 권장 전략 |
|------|----------|----------------|----------------|------------|----------|
| 基础款T恤 | 15,900 | +18% | -12% | 1.5 | 彈性大, 非必要不降价 |
| 品牌牛仔裤 | 49,900 | +8% | -6% | 0.7 | 彈性小, 可考慮漲價 |
| 季節性羽絨服 | 89,900 | +25% | -18% | 2.2 | 大促時可大幅降价 |
| 新品護膚套装 | 65,000 | +12% | -9% | 1.0 | 價值定價, 維持穩定 |
