# 성장 전략 수립 (Growth Strategy)

## 한 줄 요약
쇼핑몰의 단기/장기 성장 목표를 설정하고, 이를 달성하기 위한 실행 전략과 로드맵을 수립하는 스킬

## 상세 설명
성장 전략 수립은 쇼핑몰이 지속 가능하게 성장하기 위한 방향성과 실행 계획을 설정하는 고차원적 역량이다. 이 스킬은 시장 기회 분석, 성장 엔진 식별, 자원 배분 최적화, 리스크 관리 등을 포괄한다. AARRR(Acquisition, Activation, Retention, Referral, Revenue) 프레임워크, Growth Loop 모델, 3-Growth Strategies(-market penetration, market development, product development) 등을 활용하여 체계적인 전략을 수립한다. 또한 성장 지표(KPI) 설정, 단계별 마일스톤 정의, 애자일 방식의 실행과 피드백 루프 설계 능력이 포함된다.

## 사용 시나리오 예시

**연간 성장 전략 수립**
올해 매출 30% 성장을 목표로 할 때, 신규 고객 획득, 기존 고객 리텐션, 구매단가 향상 중 어디에 집중할지 우선순위를 정한다. 과거 데이터를 분석하여 ROI가 높은 채널을 식별하고, 투자 배분을 조정한다. 분기별 핵심 과제와 KPI를 설정하여 조직 전체가同一个 방향으로 움직이도록 한다.

**성장 정체기 돌파**
매출 성장이 정체된 쇼핑몰이 새로운 성장 동력을 확보해야 할 때, 인접 시장 진출, 신규 고객층开拓, 상품 라인 확장 등의 옵션을 평가한다. 각 옵션의 투자 대비 기대 수익, 리스크, 실행 난이도를 비교하여 최우선 과제를 선정하고, 파일럿 테스트를 통해 검증한 후 스케일업한다.

## 산출물/템플릿

| 산출물 | 내용 |
|--------|------|
| 연간 성장 전략서 | 비전, 목표, 전략 방향, 실행 과제를 포함한 종합 계획서 |
| Growth Canvas | 성장 엔진, 레버리지 포인트, 마이너스 사이클을 시각화 |
| 로드맵 | 단기/중기/장기별 핵심 이니셔티브와 마일스톤 |
| KPI 대시보드 설계서 | 핵심 지표 정의, 측정 방식, 목표값을 포함한 대시보드 설계 |

**템플릿 구조 예시: AARRR 기반 성장 전략**

```
┌─────────────────────────────────────────────────────────┐
│                    ACQUISITION (획득)                   │
│  목표: 월간 신규 고객 15% 증가                           │
│  전략: 콘텐츠 마케팅 강화, 인플루언서 협업, SEO 최적화   │
│  KPI:  CPA, 新客数, 点击率                              │
├─────────────────────────────────────────────────────────┤
│                   ACTIVATION (활성화)                   │
│  목표: 가입 7일 내 첫 구매 전환율 25%                    │
│  전략: 온보딩 경험 최적화, 웰컴 쿠폰, 상품 추천          │
│  KPI: 激活率, 第一购买时间, 浏览页面数                   │
├─────────────────────────────────────────────────────────┤
│                    RETENTION (유지)                     │
│  목표: 연간 리텐션율 45% 달성                           │
│  전략: 리워드 프로그램, 개인화 알림, 구독 서비스         │
│  KPI: 复购率, NPS, 月活跃用户数                          │
├─────────────────────────────────────────────────────────┤
│                    REFERRAL (추천)                      │
│  목표: 추천을 통한 신규 고객 비중 20%                    │
│  전략: Referral 프로그램, 후기 이벤트, 소셜 공유 Incent  │
│  KPI: K因子, 推荐转化率, 分享数                         │
├─────────────────────────────────────────────────────────┤
│                    REVENUE (매출)                       │
│  목표: 고객 생애 가치 25% 향상                           │
│  전략: 상품 추천 최적화, 번들 판매, 프리미엄 서비스      │
│  KPI: ARPU,毛利率, 平均订单价值                         │
└─────────────────────────────────────────────────────────┘
```

**성장 전략 옵션 평가 매트릭스**

| 전략 | 투자 규모 | 기대 효과 | 리스크 | 우선순위 |
|------|-----------|-----------|--------|----------|
| 市场渗透 - 现有客户深耕 | 低 | 稳定 | 低 | 1순위 |
| 市场开发 - 新客群开拓 | 中 | 高 | 中 | 2순위 |
| 产品开发 - 新品类添加 | 中 | 高 | 中高 | 3순위 |
| 多元化 - 新业务进入 | 高 | 很高 | 高 | 장기 |
| 收购并购 - 快速扩张 | 很高 | 立即 | 很高 | 조건부 |
