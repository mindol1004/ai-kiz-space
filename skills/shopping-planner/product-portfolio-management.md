# 상품 포트폴리오 관리 (Product Portfolio Management)

## 한 줄 요약
쇼핑몰의 상품 구성을 전략적으로 관리하여 매출 극대화와 재고 최적화를 달성하는 스킬

## 상세 설명
상품 포트폴리오 관리는 쇼핑몰이 보유한 상품들의 조합을 과학적으로 관리하여 수익성과 운영 효율성을 동시에 달성하는 역량이다. 이 스킬은 상품 수명주기 관리, ABC/XYZ 분석, 상품 믹스 최적화, 신상품 도입/단종 의사결정, 재고 전략 수립 등을 포함한다. 매출 기여도, 마진율, 재고 회전율을 종합하여 상품의 전략적 중요도를 분류하고, 각 범주별로 차별화된 관리 방식을 적용한다. 또한 상품 간 시너지 분석을 통한 번들링 기회 발굴, 계절말 정리 전략 등 포트폴리오 전반을 최적화하는 능력이 핵심이다.

## 사용 시나리오 예시

**분기별 상품 포트폴리오 리뷰**
분기 말 현재 취급 상품 2,000개에 대해 매출 기여도, 마진, 재고 현황을 분석한다. 매출 상위 20%가 전체 매출의 60%를 차지하는 롱테일 특성을 확인하고, 하위 30% 상품 중 재고가 과다한 품목을 식별한다. 마진율이 낮은 상품은 매입가 협상 또는 상품폐를 검토하고, 신규 트렌드 상품을 파일럿 도입한다.

**신규 브랜드 도입 결정**
트렌디한 D2C 브랜드를 신규 도입하고자 할 때, 기존 포트폴리오와의 시너지, 타겟 고객의 중복도, 마진 구조를 분석한다. 기존 브랜드와 상품 중복을 확인하고, 차별화되는 상품군을 우선 도입한다. 3개월간 테스트 판매하여 성과에 따라 본격 도입 여부를 결정한다.

## 산출물/템플릿

| 산출물 | 내용 |
|--------|------|
| 포트폴리오 현황 대시보드 | 카테고리/브랜드/가격대별 매출, 마진, 재고 현황 |
| ABC/XYZ 분석표 | 매출 기반 분류와 변동성 기반 분류를 결합한 매트릭스 |
| 수명주기 관리 계획 | 상품별 도입-성장-성숙-쇠퇴 단계별 관리 전략 |
| 상품 믹스 최적화 제안 | 추가/유지/축소 대상 상품 목록과 근거 |

**템플릿 구조 예시: ABC/XYZ 분석 매트릭스**

```
          │ 안정적 (X)       │ 정상 변동 (Y)     │ 불규칙 (Z)
────────────────────┼────────────────┼──────────────────┼─────────────────
                    │                │                  │
핵심 (A)            │ A-X: 최우선    │ A-Y: 모니터링    │ A-Z: 분석
매출 기여 상위 20%  │ 적극 관리      │ 패턴 모니터링    │ 수요 예측 강화
                    │                │                  │
                    │                │                  │
전략 (B)            │ B-X: 유지      │ B-Y: 최적화      │ B-Z: 재검토
매출 기여 중위 30%  │ 적정 재고      │ 운영 효율화      │ 주기적 재검토
                    │                │                  │
                    │                │                  │
후보 (C)            │ C-X: 고려      │ C-Y: 계절상품    │ C-Z: 단계적 폐지
매출 기여 하위 50%  │ 축소 검토      │ 시즌물 처리      │ 폐기/탈퇴
                    │                │                  │
```
