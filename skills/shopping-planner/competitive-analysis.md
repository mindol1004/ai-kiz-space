# 경쟁 분석 (Competitive Analysis)

## 한 줄 요약
경쟁사 비즈니스 모델, 전략, 강점/약점을 체계적으로 파악하여 차별화 전략을 수립하는 스킬

## 상세 설명
경쟁 분석은 시장에서 자신의 위치를 객관적으로 파악하고 지속 가능한 경쟁 우위를 확보하기 위한 필수 역량이다. 이 스킬은 경쟁사의 상품 포트폴리오, 가격 전략, 마케팅 채널, 고객 경험, 브랜드 포지셔닝 등을 다차원적으로 분석하는 것을 포함한다. 단순한 현재 상황 파악을 넘어서 경쟁사의 성장 동향, 투자 계획, 전략적 움직임을 예측하여 시사점을 도출한다. SWOT 분석, 벤치마킹 프레임워크, 경쟁 지수 산출 등 체계적인 분석 도구를 활용하며, 분석 결과를 차별화 전략과 실행 과제로 연결하는 능력이 핵심이다.

## 사용 시나리오 예시

**시장 진입 전략 수립**
건강기능식품 쇼핑몰 진입을 검토할 때, 기존 주요 경쟁사 5개사의 시장점유율, 고객층 분포,拳头产品, 마케팅 투자 규모를 분석한다. 경쟁사들의 공백地带(未满足需求)를 파악하여 차별화 포지셔닝을 설정하고, 진입 시 핵심 승부점을 도출한다.

**경쟁사 대응 전략**
대규모 할인행사를 진행하는 경쟁사의 공격적 마케팅에 대응해야 할 때, 해당 경쟁사의 할인율, 대상 상품, 채널 전략을 실시간으로 모니터링한다. 자사 고객 데이터와 비교하여 이탈 위험 고객을 식별하고, 맞춤형 프로모션을 통해 선제적으로 대응한다.

## 산출물/템플릿

| 산출물 | 내용 |
|--------|------|
| 경쟁사 프로파일 카드 | 경쟁사별 비즈니스 모델, 타겟, 강점/약점, 최근 동향을 요약한 카드 |
| 경쟁 지형 맵 | 시장점유율, 브랜드 인지도, 가격대 등을 2차원/3차원으로 시각화 |
| SWOT 비교 매트릭스 | 자사와 주요 경쟁사별 SWOT 비교 분석표 |
| 경쟁 동향 리포트 | 주요 경쟁사의 최근 전략적 움직임과 시사점 |

**템플릿 구조 예시: 경쟁 지형 맵**

```
              고가/프리미엄
                     │
        A사 (15%)    │    B사 (8%)
        ─────────────┼─────────────
        C사 (5%)     │    D사 (12%)
                     │
    ←───────────────┼──────────────→
           저가/노하우

A사: 高端有机产品, 会员体系强, 价格高
B사: 科技健康产品, 达人营销强, 新客获取强
C사: 低价基础产品, 渠道下沉深, 品牌力弱
D사: 综合保健品, 促销活动多, 复购率一般
```

**경쟁 우위 차별화 매트릭스**

|------|------|----------|----------|-------------|
