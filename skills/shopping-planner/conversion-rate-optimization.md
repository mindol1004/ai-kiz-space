# 전환율 최적화 (Conversion Rate Optimization)

## 한 줄 요약
쇼핑몰의 전환율(방문→구매)을 데이터 기반으로 지속적으로 개선하는 스킬

## 상세 설명
전환율 최적화(CRO)는 방문객이 실제 구매 고객으로 전환되는 비율을 체계적으로 개선하는 역량이다. 이 스킬은 퍼널 분석을 통한 개선 기회 식별, 가설 수립과 A/B 테스트 설계, 결과 분석과 학습 체계 구축을 포함한다. 상품 페이지 최적화(이미지, 제목, 설명), 가격 표시, CTA 버튼 최적화, 사회적 증거 활용(리뷰, 판매량), 신뢰 요소 강화(보증, 인증) 등 다양한 최적화 기법을 상황에 맞게 적용한다. 또한 모바일/PC 다른 환경에서의 차이를 고려한 개별 최적화, 그리고 전 채널 전환율 통합 관리 능력이 핵심이다.

## 사용 시나리오 예시

**A/B 테스트를 통한 CTA 최적화**
"장바구니에 담기" 버튼의 색상과 텍스트를 테스트하여 전환율 개선을 진행한다. 대조군(버튼: "담기", 색상: 파란색) 대비 실험군1("장바구니에 담기", 빨간색), 실험군2("지금 구매하기", 주황색)를 2주간 테스트한다. 실험군2가 전환율 15% 높게 나타나 대규모 적용하고, 이어서 버튼 위치 최적화를 진행한다.

**상품 페이지 최적화 프로젝트**
상품 페이지의 전환율이 평균보다 30% 낮은 카테고리를 분석한다. 상품 이미지 품질 저하, 상세 설명 불충분, 사회적 증거 부족 등의 문제가 식별된다. 먼저 고품질 이미지를 도입하고, 비디오 설명을 추가하며, 실제 고객 리뷰를 게시한다. 2주 후 해당 카테고리의 전환율이 25% 향상된다.

## 산출물/템플릿

| 산출물 | 내용 |
|--------|------|
| 전환율 대시보드 | 페이지별, 디바이스별, 채널별 전환율을 실시간 모니터링 |
| 퍼널 분석 보고서 | 각 단계별 전환율, 이탈 지점, 개선 기회 분석 |
| A/B 테스트 계획서 | 가설, 측정 지표, 샘플 크기, 기간을 포함한 테스트 계획 |
| 최적화 가이드라인 | 전환율 개선을 위한 베스트 프랙티스 모음 |

**템플릿 구조 예시: 전환율 퍼널 분석**

```
┌─────────────────────────────────────────────────────────────────┐
│ 전환율 퍼널: 방문 → 구매                                        │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│ 방문자 수                                                       │
│ 100,000 ─┐                                                      │
│          │ 전환율: 100% (모두 입장)                             │
│          ▼                                                      │
│ 첫 페이지 방문                                                  │
│ 60,000 ─┐                                                       │
│         │ 전환율: 60% (첫 페이지 이탈률 40%)                    │
│         ▼                                                       │
│ 상품 목록 페이지                                                 │
│ 36,000 ─┐                                                       │
│         │ 전환율: 60% (상품 목록 이탈 40%)                      │
│         ▼                                                       │
│ 상품 상세 페이지                                                 │
│ 21,600 ─┐                                                       │
│         │ 전환율: 60% (상품 페이지 이탈률 40%)                   │
│         ▼                                                       │
│ 장바구니 담기                                                   │
│ 10,800 ─┐                                                       │
│         │ 전환율: 50% (장바구니 담기율 50%)                     │
│         ▼                                                       │
│ 결제 페이지 이동                                                │
│ 5,400 ─┐                                                        │
│        │ 전환율: 50% (결제 페이지 이동율 50%)                   │
│        ▼                                                        │
│ 결제 완료                                                       │
│ 3,780 ─┐                                                        │
│        │ 전환율: 70% (결제 성공률 70%)                          │
│        ▼                                                        │
│ 최종 구매자 3,780 명 (전환율 3.78%)                             │
│                                                                 │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│ 개선 기회 우선순위                                              │
│ 1. 첫 페이지 → 목록: 첫 페이지 이탈률 높음, 추천 알고리즘 최적화 (영향: 높음) │
│ 2. 목록 → 상세: 상품 이미지 품질 향상 (영향: 중간)              │
│ 3. 상세 → 장바구니: 사회적 증거 강화, CTA 최적화 (영향: 높음)   │
│ 4. 장바구니 → 결제: 절차 간소화, 인센티브 제공 (영향: 중간-높음)│
│ 5. 결제 → 완료: 결제 수단 다양화 (영향: 중간)                   │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

**A/B 테스트 결과 템플릿**

| 테스트명 | 가설 | 대조군 | 실험군 | 승자 | 유의수준 | 실행 결정 |
|----------|------|--------|--------|------|----------|-----------|
| CTA 색상 | 빨간 버튼이 파란보다 전환율 높음 | 파란색 (#007bff) | 빨간색 (#dc3545) | 빨간색 | 95% | 적용 |
| CTA 문구 | "장바구니에 담기"가 "담기"보다 전환율 높음 | "담기" | "장바구니에 담기" | 동일 | - | 유지 |
| 상품 이미지 | 다중 이미지가 단일 이미지보다 우수 | 1장 | 5장 | 다중이미지 | 99% | 적용 |
| 리뷰 배치 | 별점 표시가 신뢰도 향상 | 미표시 | 상단에 표시 | 표시함 | 90% | 적용 |
