# 전환율 최적화 (Conversion Rate Optimization)

## 한 줄 요약
쇼핑몰의 전환율(방문→구매)을 데이터 기반으로 지속적으로 개선하는 스킬

## 상세 설명
전환율 최적화(CRO)는 방문객이 실제 구매 고객으로 전환되는 비율을 체계적으로 개선하는 역량이다. 이 스킬은unnel 분석을 통한 개선 기회 식별, 가설 수립과 A/B 테스트 설계, 결과 분석과 학습 체계 구축을 포함한다. 商品页面优化(图片, 标题, 描述), 价格呈现, CTA 버튼 최적化, 社会证明 활용(리뷰, 销售量), 신뢰 요소 강화(보증, 인증) 등 다양한 최적화 기법을 상황에 맞게 적용한다. 또한 Mobile/PC 다른 환경에서의 차이를 고려한 개별 최적화, 그리고 全渠道 전환율 통합 관리 능력이 핵심이다.

## 사용 시나리오 예시

**A/B 테스트를 통한 CTA 최적화**
"장바구니에 담기" 버튼의 색상과 텍스트를 테스트하여 전환율 개선을 진행한다. Control(버튼: "담기", 색상: 파란색) 대비 Test1("장바구니에 담기", 빨간색), Test2("지금 구매하기", 주황색)를 2주간 테스트한다. Test2가 전환율 15% 높게 나타나 대규모 적용하고, 이어서 버튼 위치 최적화를 진행한다.

**상품 페이지 최적화 프로젝트**
상품 페이지의 전환율이 평균보다 30% 낮은 카테고리를 분석한다. 商品图片质量低, 詳細説明不充分, 没有社会证明等问题 被识别。首先引入高质量图片，添加视频说明，并展示真实客户评论。2周后，该品类的转化率提升了25%。

## 산출물/템플릿

| 산출물 | 내용 |
|--------|------|
| 전환율 대시보드 | 페이지별, 디바이스별, 채널별 전환율을 실시간 모니터링 |
|unnel 분석 보고서 | 각 단계별 전환율, 이탈 지점, 개선 기회 분석 |
| A/B 테스트 계획서 | 가설, 측정 지표, 샘플 크기, 기간을 포함한 테스트 계획 |
| 최적화 가이드라인 | 전환율 개선을 위한 베스트 프랙티스 모음 |

```
│            ▼                                                    │
│  首页访问                                                    │
│  60,000  ─┐                                                    │
│            │  전환율: 60% (首页跳出率 40%)                      │
│            ▼                                                    │
│  商品列表页                                                   │
│  36,000  ─┐                                                    │
│            │  전환율: 60% (상품リスト離脱 40%)                  │
│            ▼                                                    │
│  商品详情页                                                   │
│  21,600  ─┐                                                    │
│            │  전환율: 60% (商品页跳出率 40%)                    │
│            ▼                                                    │
│  장바구니 담기                                                 │
│  10,800  ─┐                                                    │
│            │  전환율: 50% (加购率 50%)                          │
│            ▼                                                    │
│  결제 페이지 이동                                              │
│  5,400  ─┐                                                     │
│            │  전환율: 50% (결제페이지이동율 50%)                │
│            ▼                                                    │
│  결제 완료                                                    │
│  3,780  ─┐                                                     │
│            │  전환율: 70% (결제成功率 70%)                      │
│            ▼                                                    │
│  최종 구매자  3,780 명 (전환율 3.78%)                           │
│                                                                 │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│  개선 기회 우선순위                                            │
│  1. 首页 → 列表: 首页跳出率高, 推荐算法优化 (Impact: 高)       │
│  2. 列表 → 详情: 商品图片质量提升 (Impact: 中)                 │
│  3. 详情 → 加购: 社会证明强化, CTA优化 (Impact: 高)            │
│  4. 加购 → 支付: 简化流程, 激励措施 (Impact: 中高)             │
│  5. 支付 → 完成: 支付方式多样化 (Impact: 中)                   │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

**A/B 테스트 결과 템플릿**

| 테스트명 | 가설 | Control | Variation | 승자 | 유의수준 | 실행 결정 |
|----------|------|---------|-----------|------|----------|-----------|
| CTA颜色 | 红色按钮가蓝色보다转化率高 | 파란색 (#007bff) | 빨간색 (#dc3545) | 빨간색 | 95% | 적용 |
| CTA文案 | "장바구니에 담기"가"담기"보다转化率高 | "담기" | "장바구니에 담기" | 동일 | - | 유지 |
| 상품图片 | 多图优于单图 | 1장 | 5장 | 다중이미지 | 99% | 적용 |
| 리뷰布局 | 星级顯示提升信任 | 미표시 | 상단에 표시 | 표시함 | 90% | 적용 |
