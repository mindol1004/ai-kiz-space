# 리텐션 전략 (Retention Strategy)

## 한 줄 요약
고객의 지속적 방문과 구매를 유도하여 고객 생애 가치를 극대화하는 전략적 스킬

## 상세 설명
리텐션 전략은 신규 고객을 획득하는 것보다 기존 고객을 유지하는 것이 비용 면에서 5-25배 효율적이라는 원칙에 기반한다. 이 스킬은 고객 생애 가치(CLV) 예측, 리텐션 지표 모니터링(리피트 레이트, churn rate), 리워드/멤버십 프로그램 설계, 이탈 방지 전략, Win-back 캠페인 설계 등을 포함한다. 고객 세그먼트별 맞춤화된 유지 전략, 구매 패턴에 따른 개인화 커뮤니케이션, 고객 만족도와 충성도를 동시에 높이는 접근이 핵심이다. 또한 상품/서비스의 지속적 개선을 통해 고객의 장기적 만족을 보장한다.

## 사용 시나리오 예시

**멤버십 프로그램 리뉴얼**
기존 멤버십 프로그램의 참여율이 낮고 효과가 미미하여 전면 재설계를 진행한다. 고객 데이터를 분석하여 가장 가치 있는 등급(구매 상위 20%)을 식별하고, 이들을 위한 독점 혜택(선구매, 이벤트 초대, 무료배송)을 설계한다. 등급 간 이동 조건을 명확히 하고, 소멸 포인트를 도입하여 활동성을 유도한다. 6개월 후 멤버십 고객의 리피트 레이트가 35%에서 52%로 상승한다.

**이탈 위험 고객 대응**
최근 90일간 구매가 없는 고객 5만 명 중 구매 가능성이 높은 1만 명을 대상으로 이탈 방지 캠페인을 진행한다. 개인화된 할인 코드, 최신 인기 상품 추천, 멤버십 포인트 만료 예정 알림을 발송하고, 응답률과 재구매 전환율을 측정하여 캠페인 효과를 검증한다.

## 산출물/템플릿

| 산출물 | 내용 |
|--------|------|
| 리텐션 지표 대시보드 | RFM 분석, 리피트 레이트, Churn rate, CLV 모니터링 |
| 멤버십 프로그램 설계서 | 등급 구조, 혜택 체계, 적립/사용 규칙을 포함한 설계서 |
| 고객 세그먼트별 유지 전략 | 세그먼트별 맞춤화된 유지/이탈방지 전략 |
| Win-back 캠페인 계획서 | 이탈 고객 재활성화를 위한 캠페인 설계 |

**템플릿 구조 예시: RFM 기반 고객 세그먼트**

```
┌─────────────────────────────────────────────────────────────────┐
│ RFM 고객 세그먼트 분류                                          │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│                   │ Recency: 최근 구매일                        │
│                   │ (높음 = 최근 구매, 낮음 = 오래됨)            │
│                   │                                             │
│ 높음              │ Champions        Loyal Customers           │
│                   │ (최고 고객)      (충성 고객)                 │
│                   │ • VIP 혜택       • 로열티 프로그램          │
│ Recency           │ • 독점 이벤트    • 개인화 추천              │
│                   │ • 설문 참여      • 프리미엄 서비스           │
│                   │                                             │
│ 중간              │ Potential        At Risk Customers          │
│                   │ Loyalists        (이탈 위험)                  │
│                   │ (잠재 충성)      • 재참여 캠페인             │
│                   │ • 참여 유도      • 할인 쿠폰                 │
│                   │ • 첫 구매 혜택                                │
│                   │                                             │
│ 낮음              │ New Customers    Hibernating / Lost        │
│                   │ (신규 고객)      (휴면/이탈)                  │
│                   │ • 온보딩 강화    • Win-back 캠페인           │
│                   │ • 재구매 유도    • 마지막 기회 제안          │
│                   │                                             │
│                   └──────────────────────────────               │
│                   낮음 Frequency 높음                           │
│                   (구매 빈도)                                    │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

**리텐션 전략 매트릭스**

| 세그먼트 | 특성 | 전략 | 핵심 액션 | 기대 효과 |
|----------|------|------|-----------|-----------|
| Champions | 고빈도 고액 최근 | 유지 강화 | VIP 서비스, 전담 매니저 | CLV 극대화 |
| Loyal | 고빈도 최근 | 유지 | 포인트 리워드, 신상 우선 | 리피트 유지 |
| Potential | 저빈도 최근 | 활성화 | 구매 인센티브, 관심 추천 | 빈도 향상 |
| At Risk | 고빈도 저빈도 저액 | 회복 | 할인 혜택, 관심 연락 | 재구매 유도 |
| Hibernating | 저빈도 저액 오래됨 | Win-back | 재활성 프로모션, 특별 혜택 | 재활성화 |
| New | 신규 | 정착 | 첫 구매 경험 개선, 회원 포인트 | 2차 구매 유도 |
